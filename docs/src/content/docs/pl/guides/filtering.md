Nasze narzędzie do wizualizacji wprowadza analizowany język składni, językową magiczną różdżkę do tworzenia danych. 🚀

Illustry udostępnia kilka typowych konstrukcji i kilka konkretnych słów, aby składnia mogła zostać poprawnie przeanalizowana:

### Konstrukcje
- &&
- „>” (większy niż)
- „<” (mniej niż)
- „=” (równe)
- '>=' (większy lub równy)
- „<=” (mniejszy lub równy)
- '!=' (nierówny)
