# Wizualizacje w Twoim miejscu pracy

Podczas pracy z wizualizacjami system wspiera elastyczną strukturę danych poprzez interfejs `VisualizationData`:

## Struktura danych wizualizacji

Aby przedstawić wizualizację w swoim systemie, możesz użyć następującego interfejsu `VisualizationData`:

```typescript

{{VisualizationData}}

```

### Kluczowe atrybuty
- **projectName**: Aktywny projekt.
- **typ:** Ciąg znaków lub tablica ciągów określająca typ wizualizacji (tablica działa tylko dla JSON).
- **nazwa:** Unikalny identyfikator wizualizacji.
- **opis:** Opcjonalne pole umożliwiające podanie dodatkowego kontekstu lub szczegółów wizualizacji.
- **createdAt:** Znacznik czasu wskazujący, kiedy wizualizacja została pierwotnie utworzona.
- **updatedAt:** Znacznik czasu wskazujący ostatnią modyfikację wizualizacji.
- **isActive:** Flaga logiczna określająca, czy projekt jest aktualnie aktywny.
- **tagi:** Opcjonalne pole służące do kategoryzacji wizualizacji.
- **dane:** dane, które będzie prezentować wizualizacja
