Illustry oferuje potężny plac zabaw dla danych, pozwalający użytkownikom na interaktywne testowanie i wizualizację danych JSON przed ich zapisem. Funkcja ta jest zaprojektowana, aby zapewnić wygodny i skuteczny sposób, aby zobaczyć, jak różne wizualizacje wyglądałyby z określonego zbioru danych. Plac zabaw zawiera również mechanizm weryfikacji danych, aby zapewnić zgodność Twojego JSON z oczekiwaną strukturą.
### Rzeczywista wizualizacja czasu

Plac odtwarzania danych Illustry zapewnia wizualizację danych JSON w czasie rzeczywistym. Podczas wprowadzania lub modyfikacji danych, odpowiednie wizualizacje aktualizują dynamicznie, dając natychmiastową informację zwrotną na temat sposobu prezentacji danych.

## Jak korzystać z placu zabaw dla danych

1. Przejdź do sekcji "Data Playground" w aplikacji Illustry.
2. Wklej lub wprowadź dane JSON do dostarczonego edytora.
3. Wybierz typ wizualizacji, którą chcesz przetestować.
4. Wyświetl wizualizację w czasie rzeczywistym i w razie potrzeby dostosuj swoje dane.
5. Przeglądaj komunikaty walidacyjne, aby upewnić się, że dane są poprawnie sformatowane.
6. Zapisz swoje dane lub skopiuj zatwierdzony JSON do wykorzystania w projektach.

## Po co korzystać z placu zabaw?

- * * Visualize on the Fly: * Zobacz jak Twoje dane będą wyglądać w różnych wizualizacjach bez konieczności zapisywania lub wprowadzania zmian.
- * * Walidacja danych: * * Upewnij się, że dane JSON odpowiadają oczekiwanej strukturze, redukując błędy i usprawniając przepływ pracy.
- * * Experimentation: * * Sprawdzić różne zbiory danych i zbadać możliwości wizualizacji Illustry w środowisku wolnym od ryzyka.
