## Warunki wstępne

Przed zainstalowaniem Ilustry Frontend, upewnij się, że masz następujące warunki:

- Node wersja 18.17.0 lub wyższa
- MongoDB

## Ilusterka


#### Lokalnie:
Plik o nazwie `.env` jest wymagany w katalogu głównym projektu i powinien zawierać następujące pola:

```
MONGO_URL=mongodb:/127.0.0.1:27017/illustry
MONGO_TEST_URL=mongodb:/127.0.0.1:27017/illustrytest
MONGO_USER=<your_personal_mongo_user>
MONGO_PASSWORD=<your_personal_mongo_password>
ILLUSTRY_PORT=7000 
```
Uruchom następujące polecenia:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Doker:

Uruchom następujące polecenia:
```shell
docker build -t illustrybackend .
docker run -p 7000:7000 
```

## Frontend Illustry


#### Lokalnie:
Plik o nazwie `.env` jest wymagany w katalogu głównym projektu i powinien zawierać następujące pola:

```
NEXT_PUBLIC_BACKEND_PUBLIC_URL: http:/0.0.0.0:7000
```
Uruchom następujące polecenia:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Doker:
* * Przed uruchomieniem, stwórz najpierw obraz tła! * *

Uruchom następujące polecenia:
```shell
docker build -t illustryfrontend .
docker run -p 3000:3000 -e NEXT_PUBLIC_BACKEND_PUBLIC_URL=http:/ilustrybackend:7000 illustryfrontend
```
