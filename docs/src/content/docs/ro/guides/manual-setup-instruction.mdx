## Cerințe preliminare

Înainte de a instala Illustry Frontend, asigurați-vă că aveți următoarele cerințe preliminare:

- Node versiunea 18.17.0 sau mai recentă
- MongoDB

## Illustry backend


#### Local:
Este necesar un fișier numit `.env` la rădăcina proiectului, care conține următoarele câmpuri:

```
MONGO_URL=mongodb:/127.0.0.1:27017/illustry
MONGO_TEST_URL=mongodb:/127.0.0.1:27017/illustrytest
MONGO_USER=<your_personal_mongo_user>
MONGO_PASSWORD=<your_personal_mongo_password>
ILLUSTRY_PORT=7000 
```
Rulați următoarele comenzi:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Docher:

Rulați următoarele comenzi:
```shell
docker build -t illustrybackend .
docker run -p 7000:7000 
```

## Illustry frontend


#### Local:
Este necesar un fișier numit `.env` la rădăcina proiectului, care conține următoarele câmpuri:

```
NEXT_PUBLIC_BACKEND_PUBLIC_URL: http:/0.0.0.0:7000
```
Rulați următoarele comenzi:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Docher:
**Înainte de a rula, creați mai întâi imaginea de backend!**

Rulați următoarele comenzi:
```shell
docker build -t illustryfrontend .
docker run -p 3000:3000 -e NEXT_PUBLIC_BACKEND_PUBLIC_URL=http:/ilustrybackend:7000 illustryfrontend
```
