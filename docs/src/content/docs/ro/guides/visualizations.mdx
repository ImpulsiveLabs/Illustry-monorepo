# Vizualizări în spațiul dvs. de lucru

Când lucrați cu vizualizări, sistemul acceptă o structură de date flexibilă prin interfața `VisualizationData`:

## Structura datelor de vizualizare

Pentru a reprezenta o vizualizare în sistemul dvs., puteți utiliza următoarea interfață `VisualizationData`:

```typescript

{{VisualizationData}}

```

### Atribute cheie
- **projectName**: proiectul activ.
- **tip:** Un șir sau o matrice de șir care oferă tipul de vizualizare (matricea funcționează numai pentru JSON).
- **nume:** Un identificator unic pentru vizualizare.
- **descriere:** Un câmp opțional pentru a furniza context suplimentar sau detalii despre vizualizare.
- **createdAt:** Timpul care indică momentul în care vizualizarea a fost creată inițial.
- **updatedAt:** Timpul care indică ultima modificare a vizualizării.
- **isActive:** Un semnal boolean care specifică dacă proiectul este activ în prezent.
- **etichete:** un câmp opțional folosit pentru a clasifica vizualizările.
- **date:** Datele pe care le va prezenta vizualizarea

## Tipuri de vizualizare

Sistemul acceptă următoarele tipuri de vizualizare:

- [**WORD_CLOUD**](../word-cloud/)
- [**FORCE_DIRECTED_GRAPH**](../forced-layout-graph/)
- [**SANKEY**](../sankey/)
- [**CALENDAR**](../calendar/)
- [**HIERARCHICAL_EDGE_BUNDLING**](../hierarchical-edge-bundling/)
- [**MATRIX**](../matrix/)
- [**LINE_CHART**](../line-chart/)
- [**BAR_CHART**](../bar-chart/)
- [**PIE_CHART**](../pie-chart/)
- [**SCATTER**](../scatter/)
- [**TREEMAP**](../tree-map/)
- [**SUNBURST**](../sun-burst/)
- [**FUNNEL**](../funnel/)
- [**TIMELINE**](../timeline/)
