# לוחות מחוונים כמרחבי עבודה חזותיים

לוחות מחוונים פועלים כמרחבי עבודה ויזואליים מגוונים שבהם אתה יכול לארגן ולנהל הדמיות מרובות הקשורות לפרויקטים שלך. הם מספקים סביבה דינמית ואינטראקטיבית לחקירה, ניתוח והצגת נתונים ביעילות.

## מבנה נתונים של לוח המחוונים

כדי לייצג לוח מחוונים במערכת שלך, אתה יכול להשתמש בממשק `DashboardData` הבא:

```typescript
{{Layout}}

{{DashboardData}}

{{DashboardType}}

```
### תכונות מפתח

- **שם:** מזהה ייחודי ללוח המחוונים.
- **שם הפרויקט:** שם הפרויקט המשויך לקישור הקשרי.
- **תיאור:** שדה אופציונלי המספק הקשר או פרטים נוספים על לוח המחוונים.
- **createdAt:** חותמת הזמן המציינת מתי נוצר לוח המחוונים.
- **updatedAt:** חותמת הזמן המציגה את השינוי האחרון של לוח המחוונים.
- **הדמיות:** אוסף של הדמיות הכלולות בלוח המחוונים, כצמדי מפתח-ערך או כמערך של סוגי הדמיה.
- **פריסות:** רשימה של תצורות פריסה הקובעות את המיקום והגודל של ההדמיות בלוח המחוונים. כל פריסה כוללת:
  - **i:** מזהה ייחודי להמחשה.
  - **x, y:** קואורדינטות המייצגות את מיקום ההדמיה ברשת.
  - **w, h:** הרוחב והגובה של ההדמיה.
  - **minW, minH:** מגבלות רוחב וגובה מינימליות.

### תכונות לוח המחוונים

לוחות מחוונים מאפשרים למשתמשים:

- גרור ושחרר הדמיות כדי לסדר אותן מחדש בסביבת העבודה.
- שנה את גודל ההדמיות כדי להתאים לצרכי נתונים ספציפיים.
- שמור על פריסות מרובות כדי לתת מענה למקרי שימוש או לקהלים שונים.

על ידי מתן גמישות כזו, לוחות מחוונים מחזקים את המשתמשים להתאים אישית את סביבות העבודה החזותיות שלהם בהתאם לדרישות הפרויקט.
