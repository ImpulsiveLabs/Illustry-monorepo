# הדמיות בסביבת העבודה שלך

בעבודה עם הדמיות, המערכת תומכת במבנה נתונים גמיש באמצעות ממשק `VisualizationData`:

## מבנה נתונים להדמיה

כדי לייצג הדמיה במערכת שלך, אתה יכול להשתמש בממשק `VisualizationData` הבא:

```typescript

{{VisualizationData}}

```

### תכונות מפתח
- **שם הפרויקט**: הפרויקט הפעיל.
- **סוג:** מחרוזת או מערך מחרוזת המספקים את סוג ההדמיה (המערך עובד רק עבור JSON's).
- **שם:** מזהה ייחודי להמחשה.
- **תיאור:** שדה אופציונלי לספק הקשר נוסף או פרטים על ההדמיה.
- **createdAt:** חותמת הזמן המציינת מתי ההדמיה נוצרה לראשונה.
- **updatedAt:** חותמת הזמן המציינת את השינוי האחרון של ההדמיה.
- **isActive:** דגל בוליאני המציין אם הפרויקט פעיל כעת.
- **תגים:** שדה אופציונלי המשמש לקטגוריות ההדמיות.
- **נתונים:** הנתונים שההדמיה תציג
