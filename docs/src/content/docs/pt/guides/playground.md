Illustry oferece um poderoso Playground de dados, permitindo que os usuários testem e visualizem seus dados JSON de forma interativa antes de salvá-lo. Este recurso foi projetado para fornecer uma maneira conveniente e eficiente de ver como diferentes visualizações ficariam com seu conjunto de dados específico. O playground também inclui um mecanismo de validação de dados para garantir que seu JSON esteja em conformidade com a estrutura esperada.
### Visualização em Tempo Real

O Playground de dados Illustry fornece uma visualização em tempo real de seus dados JSON. Ao inserir ou modificar os dados, as visualizações correspondentes atualizam-se dinamicamente, dando-lhe feedback imediato sobre como seus dados serão apresentados.

## Como usar o campo de reprodução de dados

1. Navegue para a seção "Reprodução de dados" na aplicação Illustry.
2. Colar ou inserir seus dados JSON no editor fornecido.
3. Selecione o tipo de visualização que deseja testar.
4. Veja a visualização em tempo real e faça ajustes em seus dados conforme necessário.
5. Revise quaisquer mensagens de validação para garantir que seus dados sejam formatados corretamente.
6. Salve seus dados ou copie o JSON validado para uso em seus projetos.

## Por que usar o Playground de dados?

- **Visualize on the Fly:** Veja como seus dados serão vistos em diferentes visualizações sem a necessidade de salvar ou realizar alterações.
- ** Validação de dados: ** Certifique-se de que seus dados JSON aderem à estrutura esperada, reduzindo erros e simplificando seu fluxo de trabalho.
- **Experimentação: ** Teste vários conjuntos de dados e explore as capacidades das visualizações da Illustry em um ambiente sem risco.
