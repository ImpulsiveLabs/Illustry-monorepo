# Visualizações em seu espaço de trabalho

Ao trabalhar com visualizações, o sistema suporta uma estrutura de dados flexível através da interface `VisualizationData`:

## Estrutura de dados de visualização

Para representar uma visualização em seu sistema, você pode usar a seguinte interface `VisualizationData`:

```typescript

{{VisualizationData}}

```

### Atributos principais
- **projectName**: O projeto ativo.
- **type:** Uma string ou array de strings que fornece o tipo de visualização (array funciona apenas para JSON's).
- **nome:** Um identificador exclusivo para a visualização.
- **descrição:** um campo opcional para fornecer contexto ou detalhes adicionais sobre a visualização.
- **createdAt:** O carimbo de data/hora que indica quando a visualização foi criada inicialmente.
- **updatedAt:** O carimbo de data/hora que indica a última modificação da visualização.
- **isActive:** Um sinalizador booleano que especifica se o projeto está ativo no momento.
- **tags:** Um campo opcional usado para categorizar as visualizações.
- **dados:** Os dados que a visualização apresentará
