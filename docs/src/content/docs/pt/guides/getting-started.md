Illustry é uma poderosa aplicação web construída para servir como um hub de visualização, integrando de forma perfeita várias visualizações para diversos tipos de dados. A ilusão traz exploração de dados, temas e filtragem à vida. Com uma interface amigável e recursos avançados, o Illustry foi projetado para atender uma ampla gama de usuários.

## Tecnologias-chave usadas

- **Frontend:** Next 16
- **Database:** MongoDB
- **Server:** Node

## Características

- ** Integração de Dados:** Suporta vários formatos de arquivo, incluindo JSON, XML, Excel e CSV.
- ** Exploração de Dados: ** Mergulhe profundamente em seus dados com poderosas ferramentas de exploração.
- ** Theming:** Personalize a aparência e a sensação de suas visualizações com facilidade.
- **Filtragem:** Filtrar e analisar facilmente dados para extrair insights significativos.
- ** Playground:** Explore as capacidades do Illustry no playground, onde você pode interagir com os recursos e testar diferentes cenários.
