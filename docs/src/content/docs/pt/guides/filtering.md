Nossa ferramenta de visualização apresenta uma linguagem de sintaxe analisada, uma varinha mágica linguística para elaboração de dados. 🚀

Illustry fornece algumas construções comuns e algumas palavras específicas para que a sintaxe seja analisada corretamente:

### Construções
- &&
- '>' (maior que)
- '<' (menos que)
- '=' (igual)
- '>=' (maior ou igual)
- '<=' (menor ou igual)
- '!=' (diferente)
