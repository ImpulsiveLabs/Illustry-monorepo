# آپ کے کام کی جگہ میں تصورات

تصورات کے ساتھ کام کرتے وقت، نظام `VisualizationData` انٹرفیس کے ذریعے ڈیٹا کے لچکدار ڈھانچے کی حمایت کرتا ہے:

## ویژولائزیشن ڈیٹا سٹرکچر

اپنے سسٹم میں تصور کی نمائندگی کرنے کے لیے، آپ درج ذیل `VisualizationData` انٹرفیس استعمال کر سکتے ہیں:

```typescript

{{VisualizationData}}

```

### کلیدی صفات
- **پروجیکٹ کا نام**: فعال پروجیکٹ۔
- **type:** ایک سٹرنگ یا سٹرنگ اری جو تصور کی قسم فراہم کرتی ہے (صف صرف JSON's کے لیے کام کرتی ہے)۔
- **نام:** تصور کے لیے ایک منفرد شناخت کنندہ۔
- **تفصیل:** اضافی سیاق و سباق یا تصور کے بارے میں تفصیلات فراہم کرنے کے لیے ایک اختیاری فیلڈ۔
- **createdAt:** ٹائم اسٹیمپ اس بات کی نشاندہی کرتا ہے کہ ابتدائی طور پر تصور کب بنایا گیا تھا۔
- **updatedAt:** ٹائم اسٹیمپ جو ویژولائزیشن کی آخری ترمیم کی نشاندہی کرتا ہے۔
- **isActive:** ایک بولین جھنڈا یہ بتاتا ہے کہ آیا پروجیکٹ فی الحال فعال ہے۔
- **ٹیگز:** ایک اختیاری فیلڈ جو تصورات کی درجہ بندی کرنے کے لیے استعمال ہوتی ہے۔
- **ڈیٹا:** وہ ڈیٹا جو تصور پیش کرے گا۔
