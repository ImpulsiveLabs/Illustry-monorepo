ہمارا ویژولائزیشن ٹول ایک پارس شدہ نحوی زبان متعارف کراتا ہے، ڈیٹا کرافٹنگ کے لیے ایک لسانی جادوئی چھڑی۔ 🚀

Illustry کچھ عام تعمیرات اور کچھ مخصوص الفاظ فراہم کر رہا ہے تاکہ نحو کو درست طریقے سے پارس کیا جا سکے۔

### تعمیرات
- &&
- '>' (اس سے بڑا)
- '<' (اس سے کم)
- '=' (برابر)
- '>=' (اس سے بڑا یا برابر)
- '<=' (کم یا برابر)
- '!=' (برابر نہیں)
