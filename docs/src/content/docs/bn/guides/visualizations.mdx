# আপনার কর্মক্ষেত্রে ভিজ্যুয়ালাইজেশন

ভিজ্যুয়ালাইজেশনের সাথে কাজ করার সময়, সিস্টেমটি `VisualizationData` ইন্টারফেসের মাধ্যমে একটি নমনীয় ডেটা কাঠামো সমর্থন করে:

## ভিজ্যুয়ালাইজেশন ডেটা স্ট্রাকচার

আপনার সিস্টেমে একটি ভিজ্যুয়ালাইজেশন প্রতিনিধিত্ব করতে, আপনি নিম্নলিখিত `VisualizationData` ইন্টারফেস ব্যবহার করতে পারেন:

```typescript

{{VisualizationData}}

```

### মূল বৈশিষ্ট্য
- **প্রজেক্টের নাম**: সক্রিয় প্রকল্প।
- **টাইপ:** একটি স্ট্রিং বা একটি স্ট্রিং অ্যারে যা ভিজ্যুয়ালাইজেশনের ধরন প্রদান করে (অ্যারে শুধুমাত্র JSON এর জন্য কাজ করে)।
- **নাম:** ভিজ্যুয়ালাইজেশনের জন্য একটি অনন্য শনাক্তকারী।
- **বর্ণনা:** অতিরিক্ত প্রসঙ্গ বা ভিজ্যুয়ালাইজেশন সম্পর্কে বিশদ প্রদান করার জন্য একটি ঐচ্ছিক ক্ষেত্র।
- **createdAt:** টাইমস্ট্যাম্প নির্দেশ করে যখন ভিজ্যুয়ালাইজেশনটি প্রাথমিকভাবে তৈরি করা হয়েছিল।
- **আপডেটএট:** টাইমস্ট্যাম্প যা ভিজ্যুয়ালাইজেশনের শেষ পরিবর্তনকে নির্দেশ করে।
- **isActive:** একটি বুলিয়ান পতাকা নির্দিষ্ট করে যে প্রকল্পটি বর্তমানে সক্রিয় কিনা।
- **ট্যাগ:** একটি ঐচ্ছিক ক্ষেত্র যা ভিজ্যুয়ালাইজেশনকে শ্রেণীবদ্ধ করতে ব্যবহৃত হয়।
- **ডেটা:** ভিজ্যুয়ালাইজেশন যে ডেটা উপস্থাপন করবে
