Alat visualisasi kami memperkenalkan bahasa sintaksis yang diurai, tongkat ajaib linguistik untuk penyusunan data. 🚀

Illustry menyediakan beberapa konstruksi umum dan beberapa kata spesifik agar sintaksis dapat diuraikan dengan benar:

### Konstruksi
- &&
- '>' (lebih besar dari)
- '<' (kurang dari)
- '=' (sama)
- '>=' (lebih besar atau sama dengan)
- '<=' (kurang dari atau sama dengan)
- '!=' (tidak sama)
