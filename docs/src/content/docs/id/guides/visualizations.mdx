# Visualisasi di Ruang Kerja Anda

Saat bekerja dengan visualisasi, sistem mendukung struktur data yang fleksibel melalui antarmuka `VisualizationData`:

## Struktur Data Visualisasi

Untuk mewakili visualisasi di sistem Anda, Anda dapat menggunakan antarmuka `VisualizationData` berikut:

```typescript

{{VisualizationData}}

```

### Atribut Utama
- **projectName**: Proyek aktif.
- **type:** String atau array string yang menyediakan tipe visualisasi (array hanya berfungsi untuk JSON).
- **nama:** Pengenal unik untuk visualisasi.
- **deskripsi:** Bidang opsional untuk memberikan konteks atau detail tambahan tentang visualisasi.
- **createdAt:** Stempel waktu yang menunjukkan kapan visualisasi pertama kali dibuat.
- **updatedAt:** Stempel waktu yang menunjukkan modifikasi visualisasi terakhir.
- **isActive:** Tanda boolean yang menentukan apakah proyek sedang aktif.
- **tags:** Bidang opsional yang digunakan untuk mengkategorikan visualisasi.
- **data:** Data yang akan disajikan visualisasi
