## Prasyarat

Sebelum menginstal Illustry Frontend, pastikan Anda memiliki prasyarat berikut:

- Node versi 18.17.0 atau lebih tinggi
- MongoDB

## Illustry ujung belakang


#### Secara lokal:
File bernama `.env` diperlukan di root proyek yang berisi bidang berikut:

```
MONGO_URL=mongodb:/127.0.0.1:27017/illustry
MONGO_TEST_URL=mongodb:/127.0.0.1:27017/illustrytest
MONGO_USER=<your_personal_mongo_user>
MONGO_PASSWORD=<your_personal_mongo_password>
ILLUSTRY_PORT=7000 
```
Jalankan perintah berikut:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Buruh pelabuhan:

Jalankan perintah berikut:
```shell
docker build -t illustrybackend .
docker run -p 7000:7000 
```

## Illustry bagian depan


#### Secara lokal:
File bernama `.env` diperlukan di root proyek yang berisi bidang berikut:

```
NEXT_PUBLIC_BACKEND_PUBLIC_URL: http:/0.0.0.0:7000
```
Jalankan perintah berikut:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Buruh pelabuhan:
**Sebelum menjalankan, buat gambar backend terlebih dahulu!**

Jalankan perintah berikut:
```shell
docker build -t illustryfrontend .
docker run -p 3000:3000 -e NEXT_PUBLIC_BACKEND_PUBLIC_URL=http:/ilustrybackend:7000 illustryfrontend
```
