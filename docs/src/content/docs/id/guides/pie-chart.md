Visualisasi **Bagan Lingkaran** adalah cara ringkas dan berdampak visual untuk merepresentasikan distribusi bagian-bagian dalam keseluruhan. Ini sangat efektif untuk menampilkan proporsi dan persentase dalam format melingkar.


## Struktur Data Diagram Lingkaran

Untuk merepresentasikan Diagram Lingkaran, Anda dapat menggunakan antarmuka `PieChartData` berikut:

```typescript
{{PieChartData}}
```
### Atribut Utama

- **nilai:** Kamus yang setiap kuncinya merupakan kategori atau dimensi, dan nilai terkaitnya adalah nilai numerik.


### Pro dan Kontra

#### Kelebihan
- **Representasi Persentase:** Diagram Lingkaran memberikan representasi yang jelas tentang persentase kontribusi setiap kategori terhadap keseluruhan.

- **Kesederhanaan Visual:** Bentuk melingkar dan kesederhanaan Diagram Lingkaran membuatnya mudah dipahami pengguna secara sekilas.

#### Kontra
- **Terbatas untuk Banyak Kategori:** Saat menangani banyak kategori, Diagram Lingkaran bisa menjadi ramai dan sulit untuk ditafsirkan.
