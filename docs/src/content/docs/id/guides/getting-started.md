Illustry adalah aplikasi web canggih yang dibuat untuk berfungsi sebagai pusat visualisasi, yang dengan mulus mengintegrasikan berbagai visualisasi untuk beragam jenis data. Illustry menghidupkan eksplorasi, penetapan tema, dan pemfilteran data. Dengan antarmuka yang ramah pengguna dan fitur-fitur canggih, Illustry dirancang untuk melayani berbagai macam pengguna.

## Teknologi Utama yang Digunakan

- **Frontend:** Next 16
- **Database:** MongoDB
- **Server:** Node

## Fitur

- **Integrasi Data:** Mendukung berbagai format file, termasuk JSON, XML, Excel, dan CSV.
- **Eksplorasi Data:** Selidiki data Anda lebih dalam dengan alat eksplorasi canggih.
- **Tema:** Sesuaikan tampilan dan nuansa visualisasi Anda dengan mudah.
- **Pemfilteran:** Memfilter dan menganalisis data dengan mudah untuk mengekstraksi wawasan yang bermakna.
- **Taman Bermain:** Jelajahi kemampuan Illustry di taman bermain, tempat Anda dapat berinteraksi dengan fitur dan menguji berbagai skenario.
