# Trực quan hóa trong không gian làm việc của bạn

Khi làm việc với trực quan hóa, hệ thống hỗ trợ cấu trúc dữ liệu linh hoạt thông qua giao diện `VisualizationData`:

## Cấu trúc dữ liệu trực quan

Để thể hiện hình ảnh trực quan trong hệ thống của bạn, bạn có thể sử dụng giao diện `VisualizationData` sau:

```typescript

{{VisualizationData}}

```

### Thuộc tính chính
- **tên dự án**: Dự án đang hoạt động.
- **loại:** Một chuỗi hoặc một mảng chuỗi cung cấp kiểu trực quan hóa (mảng chỉ hoạt động với JSON's).
- **name:** Mã định danh duy nhất cho hình ảnh trực quan.
- **mô tả:** Trường tùy chọn để cung cấp ngữ cảnh hoặc thông tin chi tiết bổ sung về hình ảnh trực quan.
- **createdAt:** Dấu thời gian cho biết thời điểm hình ảnh trực quan được tạo lần đầu.
- **updateAt:** Dấu thời gian cho biết lần sửa đổi cuối cùng của hình ảnh trực quan.
- **isActive:** Cờ boolean xác định xem dự án hiện có đang hoạt động hay không.
- **tags:** Trường tùy chọn dùng để phân loại hình ảnh trực quan.
- **dữ liệu:** Dữ liệu mà hình ảnh trực quan sẽ trình bày
