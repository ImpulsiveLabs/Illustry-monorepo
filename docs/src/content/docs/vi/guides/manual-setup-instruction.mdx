## Người được chọn

Trước khi cài đặt Giao diện ảo tưởng, hãy đảm bảo rằng bạn có những điều kiện tiên quyết sau:

- Số nguyên số 18.17.0 hoặc cao hơn
- MongoDB

## Hậu phương ảo tưởng


#### Cục bộ:
Một tập tin tên «.env » cần thiết ở gốc của dự án chứa các trường sau:

```
MONGO_URL=mongodb:/127.0.0.1:27017/illustry
MONGO_TEST_URL=mongodb:/127.0.0.1:27017/illustrytest
MONGO_USER=<your_personal_mongo_user>
MONGO_PASSWORD=<your_personal_mongo_password>
ILLUSTRY_PORT=7000 
```
Chạy những lệnh sau:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Người đưa tin:

Chạy những lệnh sau:
```shell
docker build -t illustrybackend .
docker run -p 7000:7000 
```

## Giao diện người dùng ảo giác


#### Cục bộ:
Một tập tin tên «.env » cần thiết ở gốc của dự án chứa các trường sau:

```
NEXT_PUBLIC_BACKEND_PUBLIC_URL: http:/0.0.0.0:7000
```
Chạy những lệnh sau:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Người đưa tin:
** Trước khi chạy, tạo ảnh hậu phương trước!**

Chạy những lệnh sau:
```shell
docker build -t illustryfrontend .
docker run -p 3000:3000 -e NEXT_PUBLIC_BACKEND_PUBLIC_URL=http:/ilustrybackend:7000 illustryfrontend
```
