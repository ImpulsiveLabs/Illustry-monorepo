Візуалізація **Гістограма** — це універсальний і ефективний інструмент для представлення категоріальних даних у візуально привабливий спосіб. Він використовує горизонтальні або вертикальні смуги для відображення значень різних категорій, що полегшує користувачам порівняння та розуміння розподілу даних.

## Структура даних гістограми

Щоб представити стовпчасту діаграму, ви можете використовувати такий інтерфейс `AxisChartData`:

```typescript
{{AxisChartData}}
```
### Ключові атрибути

- **headers:** Масив рядків, що представляють категорії або розміри вздовж однієї з осей.
- **values:** Словник, де кожен ключ є категорією або розміром, а відповідне значення є масивом числових значень уздовж іншої осі.


### Плюси і мінуси

#### плюси
- **Багатовимірне порівняння:** Стовпчасті діаграми відмінно підходять для порівняння кількох вимірів одночасно, забезпечуючи повне уявлення про дані.

- **Чітке представлення значень:** Числові значення чітко представлені вздовж осей X і Y, що полегшує користувачам інтерпретацію та порівняння.

#### мінуси
- **Складність із зайвими категоріями:** Коли мова йде про велику кількість категорій або параметрів, візуалізація може стати переповненою та складною для інтерпретації.

- **Обмежено для категоріальних даних:** Гістограми є найефективнішими для числових даних і можуть бути не оптимальним вибором для представлення категоріальних даних.
