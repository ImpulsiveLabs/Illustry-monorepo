Illustry пропонує потужний ігровий майданчик даних, що дозволяє користувачам інтерактивно тестувати та візуалізувати свої дані JSON перед їх збереженням. Ця функція створена, щоб забезпечити зручний і ефективний спосіб побачити, як виглядатимуть різні візуалізації з вашим конкретним набором даних. Ігровий майданчик також містить механізм перевірки даних, щоб переконатися, що ваш JSON відповідає очікуваній структурі.
### Візуалізація в реальному часі

Illustry Data Playground забезпечує візуалізацію ваших даних JSON у реальному часі. Коли ви вводите або змінюєте дані, відповідні візуалізації динамічно оновлюються, надаючи вам негайний зворотний зв’язок щодо того, як ваші дані будуть представлені.

## Як використовувати Data Playground

1. Перейдіть до розділу «Ігровий майданчик даних» у програмі Illustry.
2. Вставте або введіть дані JSON у наданий редактор.
3. Виберіть тип візуалізації, який ви хочете перевірити.
4. Переглядайте візуалізацію в реальному часі та за потреби вносьте корективи у свої дані.
5. Перегляньте будь-які повідомлення перевірки, щоб переконатися, що ваші дані правильно відформатовані.
6. Збережіть свої дані або скопіюйте перевірений JSON для використання у своїх проектах.

## Навіщо використовувати Data Playground?

- **Візуалізація на льоту:** подивіться, як ваші дані виглядатимуть у різних візуалізаціях без необхідності зберігати чи фіксувати зміни.
- **Перевірка даних:** Переконайтеся, що ваші дані JSON відповідають очікуваній структурі, зменшуючи кількість помилок і оптимізуючи робочий процес.
- **Експеримент:** тестуйте різні набори даних і досліджуйте можливості візуалізацій Illustry у безпечному середовищі.
