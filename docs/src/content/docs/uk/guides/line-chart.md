Візуалізація **Лінійної діаграми** — це універсальний інструмент для представлення тенденцій і закономірностей у числових даних у часі. Він широко використовується для візуалізації зв’язку між двома безперервними змінними та виділення тенденцій або коливань.

## Структура даних лінійної діаграми

Щоб представити лінійну діаграму, ви можете використовувати такий інтерфейс `AxisChartData`:

```typescript
{{AxisChartData}}
```
### Ключові атрибути

- **headers:** Масив рядків, що представляють категорії або розміри вздовж однієї з осей.
- **values:** Словник, де кожен ключ є категорією або розміром, а відповідне значення є масивом числових значень уздовж іншої осі.


### Плюси і мінуси

#### плюси
- **Багатовимірне порівняння:** Лінійні діаграми чудово порівнюють кілька вимірів одночасно, забезпечуючи повне уявлення про дані.

- **Чітке представлення значень:** Числові значення чітко представлені вздовж осей X і Y, що полегшує користувачам інтерпретацію та порівняння.

#### мінуси
- **Складність із зайвими категоріями:** Коли мова йде про велику кількість категорій або параметрів, візуалізація може стати переповненою та складною для інтерпретації.

- **Обмежено для категоріальних даних:** лінійні діаграми є найефективнішими для числових даних і можуть бути не оптимальним вибором для представлення категоріальних даних.
