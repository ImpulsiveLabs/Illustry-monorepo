Наш інструмент візуалізації представляє синтаксичну мову синтаксису, лінгвістичну чарівну паличку для створення даних. 🚀

Illustry надає деякі загальні конструкції та деякі специфічні слова для правильного аналізу синтаксису:

### Конструкції
- &&
- '>' (більше ніж)
- '<' (менше)
- '=' (дорівнює)
- '>=' (більше або дорівнює)
- '<=' (менше або дорівнює)
- '!=' (не дорівнює)

## Гістограма / Лінійна діаграма

### Прийняті слова
- заголовки
- значення

## Календар

### Прийняті слова
- категорії
- дати

## Воронка/Секторна діаграма

### Прийняті слова
- значення

## Treemap / Sunburst

### Прийняті слова
- значення
- категорії

## Hierarchical-Edge-Bundling/Forced-Layout-Graph/Sankey Diagram

### Прийняті слова
- значення
- категорії
- цілі
- імена

## Скаттер

### Прийняті слова
- категорії
- xCoord
- yCoord

## Хронологія

### Прийняті слова
- види
- дати
- авторів

## Wordcloud

### Прийняті слова
- значення
