# Візуалізації у вашому робочому просторі

При роботі з візуалізаціями система підтримує гнучку структуру даних через інтерфейс `VisualizationData`:

## Структура даних візуалізації

Щоб представити візуалізацію у вашій системі, ви можете використовувати такий інтерфейс `VisualizationData`:

```typescript

{{VisualizationData}}

```

### Ключові атрибути
- **projectName**: активний проект.
- **type:** Рядок або масив рядків, який забезпечує тип візуалізації (масив працює лише для JSON).
- **name:** Унікальний ідентифікатор візуалізації.
- **опис:** необов’язкове поле для надання додаткового контексту або деталей про візуалізацію.
- **createdAt:** Мітка часу, яка вказує, коли візуалізацію було створено вперше.
- **updatedAt:** Мітка часу, що вказує на останню модифікацію візуалізації.
- **isActive:** Логічний прапор, який визначає, чи активний проект наразі.
- **теги:** необов’язкове поле, яке використовується для класифікації візуалізацій.
- **data:** Дані, які відобразить візуалізація
