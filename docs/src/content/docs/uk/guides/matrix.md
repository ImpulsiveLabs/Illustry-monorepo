Візуалізація **Матриця** — це багатовимірна візуалізація, де вузли 2 категорій можуть візуалізуватися як доповнюючі один одного

## Матрична структура даних

Щоб представити матрицю, ви можете використовувати такий інтерфейс `NodeLinkData`:

```typescript
{{Node}}

{{Link}}

{{NodeLinkData}}
```
### Ключові атрибути

- **name:** Унікальний ідентифікатор для вузла.
- **category:** класифікує вузол у певну групу або кластер.
- **мітки:** це масив значень імен, які представляють певні атрибути вузла.
- **джерело:** Ідентифікує вихідний вузол посилання.
- **target:** Ідентифікує цільовий вузол посилання.
- **value:** представляє числове значення, пов’язане з посиланням, яке вказує на те, наскільки сильно зв’язані вузли.

### Плюси і мінуси

#### плюси
- **Великі дані:** в одному вікні можна відобразити дуже великий обсяг даних.

- **Фільтрація:** Фільтрація по рядках і стовпцях.

#### мінуси
- **Потрібне прокручування:** Забагато даних може знадобитися прокручування, щоб побачити їх усі.
