Візуалізація **Діаграма Санкі** є потужним інструментом для візуалізації потоку та зв’язків між об’єктами в наборі даних. Це особливо корисно для демонстрації розподілу та трансформації цінностей у системі чи процесі.

## Структура даних діаграми Санкі

Щоб представити діаграму Санкі, ви можете використовувати такий інтерфейс `NodeLinkData`:

```typescript
{{Node}}

{{Link}}

{{NodeLinkData}}
```
### Ключові атрибути

- **name:** Унікальний ідентифікатор для вузла.
- **category:** класифікує вузол у певну групу або кластер.
- **джерело:** Ідентифікує вихідний вузол посилання.
- **target:** Ідентифікує цільовий вузол посилання.
- **value:** представляє числове значення, пов’язане з посиланням, яке вказує на те, наскільки сильно зв’язані вузли.

### Плюси і мінуси

#### плюси
- **Представлення потоку:** ефективно візуалізує потік значень або величин між об’єктами в системі або процесі.

- **Аналіз розподілу:** пропонує розуміння того, як цінності розподіляються та трансформуються на кожному етапі.

#### мінуси
- **Складність із великою кількістю вузлів: ** Візуальний безлад може виникати з великою кількістю вузлів, що впливає на читабельність.

- **Спеціалізований варіант використання:** Головним чином призначений для візуалізації потоку та розподілу; може не підходити для всіх типів наборів даних.
