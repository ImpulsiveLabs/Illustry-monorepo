**선 차트** 시각화는 시간 경과에 따른 숫자 데이터의 추세와 패턴을 나타내는 다목적 도구입니다. 이는 두 개의 연속 변수 간의 관계를 시각화하고 추세 또는 변동을 강조하는 데 널리 사용됩니다.

## 꺾은선형 차트 데이터 구조

꺾은선형 차트를 나타내려면 다음 `AxisChartData` 인터페이스를 사용할 수 있습니다.

```typescript
{{AxisChartData}}
```
### 주요 속성

- **헤더:** 축 중 하나를 따라 카테고리 또는 측정기준을 나타내는 문자열 배열입니다.
- **값:** 각 키가 카테고리 또는 측정기준이고 해당 값이 다른 축을 따른 숫자 값 배열인 사전입니다.


### 장점과 단점

#### 장점
- **다차원 비교:** 선 차트는 여러 차원을 동시에 비교하는 데 탁월하여 데이터에 대한 포괄적인 보기를 제공합니다.

- **명확한 값 표시:** 숫자 값은 X축과 Y축 모두에 명확하게 표시되므로 사용자가 쉽게 해석하고 비교할 수 있습니다.

#### 단점
- **과도한 카테고리로 인한 복잡성:** 많은 수의 카테고리나 차원을 처리할 때 시각화가 복잡해지고 해석하기 어려울 수 있습니다.

- **범주형 데이터에 대해 제한됨:** 꺾은선형 차트는 숫자 데이터에 가장 효과적이며 범주형 데이터를 나타내는 데 최적의 선택이 아닐 수도 있습니다.
