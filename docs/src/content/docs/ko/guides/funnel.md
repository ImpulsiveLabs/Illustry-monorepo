**깔때기** 시각화는 프로세스 내 일련의 단계의 진행률과 전환율을 추적하기 위한 강력한 도구입니다. 정의된 워크플로의 여러 단계를 통해 엔터티가 이동하는 방식을 시각적으로 표현합니다.



## 유입경로 데이터 구조

유입경로를 나타내려면 다음 `FunnelData` 인터페이스를 사용할 수 있습니다.

```typescript
{{FunnelData}}
```
### 주요 속성

- **값:** 각 키가 카테고리 또는 측정기준이고 해당 값이 숫자 값인 사전입니다.


### 장점과 단점

#### 장점
- **성과 지표:** 깔때기형 차트는 성과 지표를 시각화하는 데 탁월하여 팀이 주요 지표를 추적하고 개선할 수 있도록 해줍니다.

- **세분화된 분석:** 유입경로의 각 단계에서는 세분화된 분석이 가능하여 개선 또는 성공의 특정 영역을 식별하는 데 도움이 됩니다.

- **예측 통찰력:** 유입경로 차트는 과거 전환율을 기반으로 미래 실적에 대한 예측 통찰력을 제공할 수 있습니다.

#### 단점

- **복잡한 프로세스로 인한 어려움:** 프로세스에 수많은 복잡한 분기 또는 루프가 포함된 시나리오에서는 깔때기형 차트가 복잡성을 효과적으로 표현하는 데 어려움을 겪을 수 있습니다.
