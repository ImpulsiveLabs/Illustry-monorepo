**산점도** 시각화는 두 수치 변수 간의 관계를 시각화하는 강력한 도구입니다. 이는 데카르트 평면의 점을 사용하여 개별 데이터 점을 나타내므로 패턴, 상관 관계 및 이상값을 쉽게 식별할 수 있습니다.


## 산점도 데이터 구조

산점도를 나타내려면 다음 `ScatterData` 인터페이스를 사용할 수 있습니다.

```typescript
{{ScatterPoint}}

{{ScatterData}}
```

### 주요 속성

- **값:** X 및 Y축의 데이터 포인트 좌표를 나타내는 두 숫자 값 [x, y]의 배열입니다.
- **값:** 데이터 포인트가 속한 카테고리 또는 그룹을 나타내는 문자열입니다.


### 장점과 단점

#### 장점
- **관계 식별:** 산점도는 두 수치 변수 간의 관계, 패턴 및 추세를 나타내는 데 탁월합니다.

- **이상치 감지:** 표준에서 크게 벗어난 이상치 또는 데이터 포인트는 산점도에서 쉽게 식별됩니다.

#### 단점

- **잠재적인 과도표:** 데이터 포인트의 밀도가 높은 경우 과도표가 발생하여 개별 포인트의 가시성에 영향을 미칠 수 있습니다.
