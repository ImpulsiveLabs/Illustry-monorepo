Illustry은 시각화 허브 역할을 하도록 구축된 강력한 웹 애플리케이션으로, 다양한 유형의 데이터에 대한 다양한 시각화를 원활하게 통합합니다. Illustry에서는 데이터 탐색, 테마 지정, 필터링이 생생하게 구현됩니다. 사용자 친화적인 인터페이스와 고급 기능을 갖춘 Illustry은(는) 광범위한 사용자를 수용하도록 설계되었습니다.

## 사용된 주요 기술

- **Frontend:** Next 16
- **Database:** MongoDB
- **Server:** Node

## 특징

- **데이터 통합:** JSON, XML, Excel 및 CSV을 포함한 다양한 파일 형식을 지원합니다.
- **데이터 탐색:** 강력한 탐색 도구를 사용하여 데이터를 자세히 살펴보세요.
- **테마 지정:** 시각화의 모양과 느낌을 쉽게 맞춤설정하세요.
- **필터링:** 데이터를 쉽게 필터링하고 분석하여 의미 있는 통찰력을 추출합니다.
- **플레이그라운드:** 플레이그라운드에서 Illustry의 기능을 살펴보고 기능과 상호 작용하고 다양한 시나리오를 테스트할 수 있습니다.
