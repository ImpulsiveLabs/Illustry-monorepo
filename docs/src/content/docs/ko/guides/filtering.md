우리의 시각화 도구는 데이터 제작을 위한 언어적 마술 지팡이인 구문 분석된 구문 언어를 도입합니다. 🚀

Illustry은 구문이 올바르게 구문 분석될 수 있도록 몇 가지 일반적인 구문과 몇 가지 특정 단어를 제공합니다.

### 건축
- &&
- '>'(보다 큼)
- '<'(보다 작음)
- '='(같음)
- '>='(크거나 같음)
- '<='(작거나 같음)
- '!='(같지 않음)

## 막대 차트/선 차트

### 허용되는 단어
- 헤더
- 가치

## 달력

### 허용되는 단어
- 카테고리
- 날짜

## 유입경로/원형 차트

### 허용되는 단어
- 가치

## 트리맵/선버스트

### 허용되는 단어
- 가치
- 카테고리

## 계층적-에지-번들링/강제 레이아웃-그래프/Sankey 다이어그램

### 허용되는 단어
- 가치
- 카테고리
- 목표
- 이름

## 소수

### 허용되는 단어
- 카테고리
- xCoord
- y좌표

## 타임라인

### 허용되는 단어
- 종류
- 날짜
- 작가

## 워드클라우드

### 허용되는 단어
- 가치
