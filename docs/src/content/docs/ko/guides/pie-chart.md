**원형 차트** 시각화는 전체 내에서 부분의 분포를 표현하는 간결하고 시각적으로 효과적인 방법입니다. 원형 형식으로 비율과 백분율을 표시하는 데 특히 효과적입니다.


## 원형 차트 데이터 구조

원형 차트를 나타내려면 다음 `PieChartData` 인터페이스를 사용할 수 있습니다.

```typescript
{{PieChartData}}
```
### 주요 속성

- **값:** 각 키가 카테고리 또는 측정기준이고 해당 값이 숫자 값인 사전입니다.


### 장점과 단점

#### 장점
- **백분율 표시:** 원형 차트는 전체에 대한 각 카테고리의 기여도를 백분율로 명확하게 표현합니다.

- **시각적 단순성:** 원형 차트의 단순성과 원형 덕분에 사용자는 한눈에 쉽게 이해할 수 있습니다.

#### 단점
- **다양한 카테고리에 대해 제한됨:** 수많은 카테고리를 처리할 때 원형 차트는 복잡해지고 해석하기 어려울 수 있습니다.
