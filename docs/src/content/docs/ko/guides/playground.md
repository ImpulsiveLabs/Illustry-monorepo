Illustry은(는) 강력한 데이터 놀이터를 제공하므로 사용자는 JSON 데이터를 저장하기 전에 대화형으로 테스트하고 시각화할 수 있습니다. 이 기능은 특정 데이터세트에서 다양한 시각화가 어떻게 보이는지 확인할 수 있는 편리하고 효율적인 방법을 제공하도록 설계되었습니다. 플레이그라운드에는 JSON이 예상 구조를 준수하는지 확인하는 데이터 검증 메커니즘도 포함되어 있습니다.
### 실시간 시각화

Illustry 데이터 놀이터는 JSON 데이터의 실시간 시각화를 제공합니다. 데이터를 입력하거나 수정하면 해당 시각화가 동적으로 업데이트되어 데이터가 표시되는 방식에 대한 즉각적인 피드백을 제공합니다.

## 데이터 놀이터 이용 방법

1. Illustry 애플리케이션의 "Data Playground" 섹션으로 이동합니다.
2. 제공된 편집기에 JSON 데이터를 붙여넣거나 입력하세요.
3. 테스트하려는 시각화 유형을 선택합니다.
4. 실시간 시각화를 확인하고 필요에 따라 데이터를 조정하세요.
5. 유효성 검사 메시지를 검토하여 데이터 형식이 올바른지 확인하세요.
6. 프로젝트에 사용할 수 있도록 데이터를 저장하거나 검증된 JSON을 복사하세요.

## 데이터 놀이터를 사용하는 이유는 무엇입니까?

- **즉시 시각화:** 변경 사항을 저장하거나 커밋할 필요 없이 데이터가 다양한 시각화에서 어떻게 보이는지 확인하세요.
- **데이터 검증:** JSON 데이터가 예상 구조를 준수하는지 확인하여 오류를 줄이고 작업 흐름을 간소화합니다.
- **실험:** 다양한 데이터세트를 테스트하고 위험이 없는 환경에서 Illustry의 시각화 기능을 살펴보세요.
