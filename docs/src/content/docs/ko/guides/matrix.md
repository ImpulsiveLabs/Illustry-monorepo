**매트릭스** 시각화는 2개 범주의 노드를 다른 범주와 보완적으로 시각화할 수 있는 다차원 시각화입니다.

## 매트릭스 데이터 구조

매트릭스를 나타내기 위해 다음 `NodeLinkData` 인터페이스를 사용할 수 있습니다.

```typescript
{{Node}}

{{Link}}

{{NodeLinkData}}
```
### 주요 속성

- **이름:** 노드의 고유 식별자입니다.
- **범주:** 노드를 특정 그룹이나 클러스터로 분류합니다.
- **레이블:** 노드의 특정 속성을 나타내는 이름 값의 배열입니다.
- **source:** 링크의 소스 노드를 식별합니다.
- **target:** 링크의 대상 노드를 식별합니다.
- **값:** 노드가 얼마나 강하게 연결되어 있는지를 나타내는 링크와 관련된 숫자 값을 나타냅니다.

### 장점과 단점

#### 장점
- **대량 데이터:** 매우 많은 양의 데이터를 단일 창에 표시할 수 있습니다.

- **필터링:** 행과 열을 필터링합니다.

#### 단점
- **스크롤 필요:** 데이터가 너무 많으면 모두 보려면 스크롤이 필요할 수 있습니다.
