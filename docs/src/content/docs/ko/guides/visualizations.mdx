# 작업 공간의 시각화

시각화 작업 시 시스템은 `VisualizationData` 인터페이스를 통해 유연한 데이터 구조를 지원합니다.

## 시각화 데이터 구조

시스템에서 시각화를 나타내려면 다음 `VisualizationData` 인터페이스를 사용할 수 있습니다.

```typescript

{{VisualizationData}}

```

### 주요 속성
- **projectName**: 활성 프로젝트입니다.
- **유형:** 시각화 유형을 제공하는 문자열 또는 문자열 배열입니다(배열은 JSON에 대해서만 작동합니다).
- **이름:** 시각화의 고유 식별자입니다.
- **설명:** 시각화에 대한 추가 컨텍스트나 세부정보를 제공하는 선택적 필드입니다.
- **createdAt:** 시각화가 처음 생성된 시기를 나타내는 타임스탬프입니다.
- **updatedAt:** 시각화의 마지막 수정을 나타내는 타임스탬프입니다.
- **isActive:** 프로젝트가 현재 활성화되어 있는지 여부를 지정하는 부울 플래그입니다.
- **태그:** 시각화를 분류하는 데 사용되는 선택적 필드입니다.
- **데이터:** 시각화가 표시할 데이터입니다.

시스템은 다음 시각화 유형을 지원합니다.

- [**FORCE_DIRECTED_GRAPH**](../forced-layout-graph/)
