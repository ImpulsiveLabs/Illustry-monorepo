## Vereisten

Voordat u Illustry Frontend installeert, moet u ervoor zorgen dat u aan de volgende vereisten voldoet:

- Knooppunt versie 18.17.0 of hoger
- MongoDB

## Illustry backend


#### Lokaal:
Er is een bestand met de naam `.env` nodig in de hoofdmap van het project dat de volgende velden bevat:

```
MONGO_URL=mongodb:/127.0.0.1:27017/illustry
MONGO_TEST_URL=mongodb:/127.0.0.1:27017/illustrytest
MONGO_USER=<your_personal_mongo_user>
MONGO_PASSWORD=<your_personal_mongo_password>
ILLUSTRY_PORT=7000 
```
Voer de volgende opdrachten uit:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Dokwerker:

Voer de volgende opdrachten uit:
```shell
docker build -t illustrybackend .
docker run -p 7000:7000 
```

## Illustry frontend


#### Lokaal:
Er is een bestand met de naam `.env` nodig in de hoofdmap van het project dat de volgende velden bevat:

```
NEXT_PUBLIC_BACKEND_PUBLIC_URL: http:/0.0.0.0:7000
```
Voer de volgende opdrachten uit:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Dokwerker:
**Maak eerst de backend-image voordat u begint!**

Voer de volgende opdrachten uit:
```shell
docker build -t illustryfrontend .
docker run -p 3000:3000 -e NEXT_PUBLIC_BACKEND_PUBLIC_URL=http:/ilustrybackend:7000 illustryfrontend
```
