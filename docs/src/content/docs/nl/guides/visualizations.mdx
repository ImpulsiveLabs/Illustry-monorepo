# Visualisaties in uw werkruimte

Bij het werken met visualisaties ondersteunt het systeem een ​​flexibele datastructuur via de `VisualizationData` interface:

## Visualisatiegegevensstructuur

Om een ​​visualisatie in uw systeem weer te geven, kunt u de volgende `VisualizationData` interface gebruiken:

```typescript

{{VisualizationData}}

```

### Belangrijkste kenmerken
- **projectName**: het actieve project.
- **type:** Een string of een stringarray die het type visualisatie levert (array werkt alleen voor JSON's).
- **naam:** Een unieke identificatie voor de visualisatie.
- **beschrijving:** Een optioneel veld om aanvullende context of details over de visualisatie te bieden.
- **createdAt:** Het tijdstempel dat aangeeft wanneer de visualisatie voor het eerst is gemaakt.
- **updatedAt:** De tijdstempel die de laatste wijziging van de visualisatie aangeeft.
- **isActive:** Een Booleaanse vlag die aangeeft of het project momenteel actief is.
- **tags:** Een optioneel veld dat wordt gebruikt om de visualisaties te categoriseren.
- **data:** De gegevens die de visualisatie zal presenteren
