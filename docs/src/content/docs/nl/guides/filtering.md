Onze visualisatietool introduceert een ontlede syntaxistaal, een taalkundige toverstaf voor het bewerken van gegevens. 🚀

Illustry biedt enkele algemene constructies en enkele specifieke woorden zodat de syntaxis correct kan worden ontleed:

### Constructies
- &&
- '>' (groter dan)
- '<' (minder dan)
- '=' (gelijk)
- '>=' (groter dan of gelijk)
- '<=' (kleiner dan of gelijk aan)
- '!=' (niet gelijk)
