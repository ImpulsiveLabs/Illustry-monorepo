# 工作空间中的可视化

使用可视化时，系统通过 `VisualizationData` 接口支持灵活的数据结构：

## 可视化数据结构

要在系统中表示可视化，您可以使用以下 `VisualizationData` 接口：

```typescript

{{VisualizationData}}

```

### 关键属性
- **项目名称**：活动项目。
- **类型：** 提供可视化类型的字符串或字符串数​​组（数组仅适用于 JSON's）。
- **名称：** 可视化的唯一标识符。
- **描述：** 可选字段，用于提供有关可视化的其他上下文或详细信息。
- **createdAt：** 指示可视化最初创建时间的时间戳。
- **updatedAt：** 指示可视化的最后一次修改的时间戳。
- **isActive：** 一个布尔标志，指定项目当前是否处于活动状态。
- **标签：** 用于对可视化进行分类的可选字段。
- **数据：** 可视化将呈现的数据
