我们的可视化工具引入了解析语法语言，这是用于数据制作的语言魔杖。 🚀

Illustry 提供了一些常见的结构和一些特定的单词，以便正确解析语法：

### 建筑
- &&
- '>'（大于）
- '<'（小于）
- '='（等于）
- '>='（大于或等于）
- '<='（小于或等于）
- '!='（不等于）

## 条形图/折线图

### 接受的词
- 标头
- 价值观

## 日历

### 接受的词

- 日期

## 漏斗图/饼图

### 接受的词

- 名字
