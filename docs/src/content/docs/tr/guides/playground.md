## Veri Oyun Alanı Nasıl Sayılır

1. Illustry "Veri Oyun Alanı" ETF'si.
2. JSON verilerinize verilen editöre giriş yapın.
3. Test etmek istediğiniz görselleştirme türünü seçin.
4. Gerçek zamanlı görselleştirmeyi görün ve gerekli olduğu gibi verilerinize ayarlamalar yapın.
5. Verilerinizin doğru şekilde formatlanmış olmasını sağlamak için geçerli olan mesajları gözden geçirin.
6. Verilerinizi kurtar veya projelerinizde kullanılmak için doğrulanan JSON'u kopyalayın.
