# تصورات في مساحة العمل الخاصة بك

عند العمل مع المرئيات، يدعم النظام بنية بيانات مرنة من خلال واجهة `VisualizationData`:

## هيكل بيانات التصور

لتمثيل تصور في نظامك، يمكنك استخدام واجهة `VisualizationData` التالية:

```typescript

{{VisualizationData}}

```

### السمات الرئيسية
- **اسم المشروع**: المشروع النشط.
- **النوع:** سلسلة أو مصفوفة سلسلة توفر نوع التصور (المصفوفة تعمل فقط مع JSON's).
- **الاسم:** معرف فريد للتصور.
- **الوصف:** حقل اختياري لتوفير سياق إضافي أو تفاصيل حول التمثيل المرئي.
- **createdAt:** الطابع الزمني الذي يشير إلى وقت إنشاء التمثيل المرئي في البداية.
- **updatedAt:** الطابع الزمني الذي يشير إلى التعديل الأخير للتصور.
- **isActive:** علامة منطقية تحدد ما إذا كان المشروع نشطًا حاليًا.
- **العلامات:** حقل اختياري يستخدم لتصنيف المرئيات.
- **البيانات:** البيانات التي سيقدمها التصور
