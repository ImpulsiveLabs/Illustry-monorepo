تقدم أداة التصور الخاصة بنا لغة تركيبية مُحللة، وهي عصا لغوية سحرية لصياغة البيانات. 🚀

يقدم Illustry بعض الإنشاءات الشائعة وبعض الكلمات المحددة حتى يتم تحليل بناء الجملة بشكل صحيح:

### الانشاءات
- &&
- '>' (أكبر من)
- '<' (أقل من)
- '=' (يساوي)
- '>=' (أكبر من أو يساوي)
- '<=' (أقل من أو يساوي)
- '!=' (غير متساوي)
