Illustry は強力なデータ プレイグラウンドを提供し、ユーザーは JSON データを保存する前に対話的にテストして視覚化できます。この機能は、特定のデータセットでさまざまなビジュアライゼーションがどのように見えるかを確認する便利で効率的な方法を提供するように設計されています。プレイグラウンドには、JSON が予想される構造に準拠していることを確認するためのデータ検証メカニズムも含まれています。
### リアルタイムの可視化

Illustry データ プレイグラウンドは、JSON データのリアルタイムの視覚化を提供します。データを入力または変更すると、対応するビジュアライゼーションが動的に更新され、データがどのように表示されるかについて即座にフィードバックが得られます。

## データ プレイグラウンドの使用方法

1. Illustry アプリケーションの「データ プレイグラウンド」セクションに移動します。
2. JSON データを提供されたエディターに貼り付けるか入力します。
3. テストするビジュアライゼーションのタイプを選択します。
4. リアルタイムの視覚化を表示し、必要に応じてデータを調整します。
5. 検証メッセージを確認して、データが正しくフォーマットされていることを確認します。
6. プロジェクトで使用するためにデータを保存するか、検証された JSON をコピーします。

## データ プレイグラウンドを使用する理由

- **オンザフライで視覚化:** 変更を保存したりコミットしたりすることなく、さまざまな視覚化でデータがどのように表示されるかを確認できます。
- **データ検証:** JSON データが予想される構造に従っていることを確認し、エラーを減らし、ワークフローを合理化します。
- **実験:** さまざまなデータセットをテストし、リスクのない環境で Illustry のビジュアライゼーションの機能を調査します。
