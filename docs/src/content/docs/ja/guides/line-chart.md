**折れ線グラフ** ビジュアライゼーションは、時間の経過に伴う数値データの傾向とパターンを表す多用途ツールです。 2 つの連続変数間の関係を視覚化し、傾向や変動を強調するために広く使用されています。

## 折れ線グラフのデータ構造

折れ線グラフを表すには、次の `AxisChartData` インターフェイスを使用できます。

```typescript
{{AxisChartData}}
```
### 主要な属性

- **headers:** いずれかの軸に沿ったカテゴリまたはディメンションを表す文字列の配列。
- **値:** 各キーがカテゴリまたはディメンションであり、対応する値がもう一方の軸に沿った数値の配列であるディクショナリ。


### 長所と短所

#### 長所
- **多次元比較:** 折れ線グラフは、複数の次元を同時に比較することに優れており、データの包括的なビューを提供します。

- **明確な値の表現:** 数値は X 軸と Y 軸の両方に沿って明確に表現されるため、ユーザーは解釈と比較が容易になります。

#### 短所
- **過剰なカテゴリによる複雑さ:** 多数のカテゴリまたはディメンションを扱う場合、ビジュアライゼーションが混雑して解釈が難しくなる可能性があります。

- **カテゴリ データに限定:** 折れ線グラフは数値データに最も効果的ですが、カテゴリ データを表すには最適な選択ではない場合があります。
