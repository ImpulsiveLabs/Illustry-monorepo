Illustry は、ユーザーがビジュアライゼーションの外観と操作性を調整できる強力なテーマのカスタマイズ機能を提供します。ダーク モードとライト モードのどちらを好む場合でも、事前定義されたテーマから選択したり、視覚化の種類ごとに色を微調整したりすることができます。

## 事前定義されたテーマ

Illustry には、ワークスペースに簡単に適用できる事前定義されたテーマのセットが付属しています。これらのテーマは、調和のとれた視覚的に魅力的なエクスペリエンスを保証するように作成されています。テーマを簡単に切り替えて、好みに合ったテーマを見つけることができます。

## カスタムテーマの編集

ビジュアライゼーションの美しさをより詳細に制御したいユーザーのために、Illustry はカスタム テーマ編集機能を提供します。これにより、視覚化の各タイプのさまざまな要素に特定の色を選択できます。背景色、グラフの色合い、テキストの色合いを調整する場合でも、創造的な自由を持ってワークスペースを完璧にデザインできます。

## ダークモードとライトモード

Illustry はダーク モードとライト モードの両方をサポートし、さまざまな好みに応え、さまざまな環境で最適な視認性を確保します。モードをシームレスに切り替えて作業条件に適応し、全体的なユーザー エクスペリエンスを向上させることができます。

## テーマ機能へのアクセス方法

1. Illustry ダッシュボードの「テーマ」セクションに移動します。
2. 事前定義されたテーマを選択するか、カスタム テーマ エディターを入力します。
3. テーマに設定するビジュアライゼーションのタイプを選択します。
4. 好みに応じて色や設定を調整します。
