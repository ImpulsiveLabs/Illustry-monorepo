**サンキー ダイアグラム** 視覚化は、データセット内のエンティティ間のフローと関係を視覚化するための強力なツールです。これは、システムまたはプロセス全体にわたる値の分布と変換を示す場合に特に役立ちます。

## サンキーダイアグラムのデータ構造

サンキー ダイアグラムを表すには、次の `NodeLinkData` インターフェイスを使用できます。

```typescript
{{Node}}

{{Link}}

{{NodeLinkData}}
```
### 主要な属性

- **name:** ノードの一意の識別子。
- **category:** ノードを特定のグループまたはクラスターに分類します。
- **source:** リンクのソース ノードを識別します。
- **target:** リンクのターゲット ノードを識別します。
- **値:** ノードの接続の強さを示す、リンクに関連付けられた数値を表します。

### 長所と短所

#### 長所
- **フロー表現:** システムまたはプロセス内のエンティティ間の値または数量の流れを効果的に視覚化します。

- **分布に関する洞察:** 各段階で価値がどのように分布および変換されるかについての洞察を提供します。

#### 短所
- **多数のノードによる複雑さ:** 多数のノードがあると視覚的に乱雑な状態が発生し、可読性に影響を与える可能性があります。

- **特殊な使用例:** 主にフローと分布を視覚化するために設計されています。すべてのタイプのデータセットに適しているとは限りません。
