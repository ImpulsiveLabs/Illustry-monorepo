---

Illustry は、視覚化ハブとして機能するように構築された強力な Web アプリケーションであり、さまざまな種類のデータのさまざまな視覚化をシームレスに統合します。 Illustry は、データ探索、テーマ設定、フィルタリングを実現します。ユーザーフレンドリーなインターフェイスと高度な機能を備えた Illustry は、幅広いユーザーに対応できるように設計されています。

## 使用されている主要なテクノロジー

- **Frontend:** Next 16
- **Database:** MongoDB
- **Server:** Node

## 特徴

- **データ統合:** JSON、XML、Excel、CSV などのさまざまなファイル形式をサポートします。
- **データ探索:** 強力な探索ツールを使用してデータを深く掘り下げます。
- **テーマ:** ビジュアライゼーションの外観と雰囲気を簡単にカスタマイズします。
- **フィルタリング:** データを簡単にフィルタリングおよび分析して、有意義な洞察を抽出します。
- **プレイグラウンド:** プレイグラウンドで Illustry の機能を調べて、機能を操作したり、さまざまなシナリオをテストしたりできます。
