**ファネル** ビジュアライゼーションは、プロセスの一連の段階の進行状況とコンバージョン率を追跡するための強力なツールです。これは、エンティティが定義されたワークフローのさまざまな段階をどのように移動するかを視覚的に表現します。



## ファネルのデータ構造

ファネルを表すには、次の `FunnelData` インターフェイスを使用できます。

```typescript
{{FunnelData}}
```
### 主要な属性

- **値:** 各キーがカテゴリまたはディメンションであり、対応する値が数値であるディクショナリ。


### 長所と短所

#### 長所
- **パフォーマンス メトリクス:** ファネル チャートはパフォーマンス メトリクスの視覚化に優れており、チームが主要な指標を追跡および改善できるようになります。

- **セグメント化された分析:** ファネルの各段階ではセグメント化された分析が可能であり、改善や成功の特定の領域を特定するのに役立ちます。

- **予測的洞察:** ファネル チャートは、過去のコンバージョン率に基づいて将来のパフォーマンスに対する予測的な洞察を提供します。

#### 短所

- **複雑なプロセスに関する課題:** プロセスに多数の複雑な分岐やループが含まれるシナリオでは、ファネル チャートは複雑さを効果的に表現するのに苦労する可能性があります。
