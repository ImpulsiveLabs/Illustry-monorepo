**散布図** 視覚化は、2 つの数値変数間の関係を視覚化するための強力なツールです。デカルト平面上の点を使用して個々のデータ ポイントを表すため、パターン、相関関係、および外れ値を簡単に識別できます。


## 散布図のデータ構造

散布図を表すには、次の `ScatterData` インターフェイスを使用できます。

```typescript
{{ScatterPoint}}

{{ScatterData}}
```

### 主要な属性

- **値:** X 軸と Y 軸上のデータ点の座標を表す 2 つの数値 [x, y] の配列。
- **値:** データ ポイントが属するカテゴリまたはグループを示す文字列。


### 長所と短所

#### 長所
- **関係の特定:** 散布図は、2 つの数値変数間の関係、パターン、傾向を明らかにすることに優れています。

- **外れ値の検出:** 外れ値、つまり標準から大きく逸脱したデータ ポイントは、散布図上で簡単に特定できます。

#### 短所

- **オーバープロットの可能性:** データ ポイントの密度が高い場合、オーバープロットが発生し、個々のポイントの可視性に影響を与える可能性があります。
