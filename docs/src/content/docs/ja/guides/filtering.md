当社の視覚化ツールは、データ クラフトのための言語魔法の杖である解析構文言語を導入しています。 🚀

Illustry は、構文が正しく解析されるように、いくつかの一般的な構造といくつかの特定の単語を提供します。

### 構造物
- &&
- '>' (より大きい)
- '<' (より小さい)
- '=' (等しい)
- '>=' (以上)
- '<=' (以下)
- '!=' (等しくない)

## 棒グラフ/折れ線グラフ

### 受け入れられる単語
- ヘッダー
- 価値観

## カレンダー

### 受け入れられる単語
- カテゴリ
- 日付

## ファネル/円グラフ

### 受け入れられる単語
- 価値観

## ツリーマップ / サンバースト

### 受け入れられる単語
- 価値観
- カテゴリ

## 階層エッジバンドル / 強制レイアウトグラフ / サンキーダイアグラム

### 受け入れられる単語
- 価値観
- カテゴリ
- ターゲット
- 名前

## 散布

### 受け入れられる単語
- カテゴリ
- xCoord
- y座標

## タイムライン

### 受け入れられる単語
- 種類

- 著者

## ワードクラウド

### 受け入れられる単語
- 価値観
