# ワークスペースのビジュアライゼーション

ビジュアライゼーションを操作する場合、システムは `VisualizationData` インターフェイスを通じて柔軟なデータ構造をサポートします。

## 可視化データ構造

システム内でビジュアライゼーションを表現するには、次の `VisualizationData` インターフェイスを使用できます。

```typescript

{{VisualizationData}}

```

### 主要な属性
- **projectName**: アクティブなプロジェクト。
- **type:** 視覚化のタイプを提供する文字列または文字列配列 (配列は JSON でのみ機能します)。
- **name:** ビジュアライゼーションの一意の識別子。
- **説明:** ビジュアライゼーションに関する追加のコンテキストまたは詳細を提供するオプションのフィールド。
- **createdAt:** ビジュアライゼーションが最初に作成された時間を示すタイムスタンプ。
- **updatedAt:** ビジュアライゼーションの最終変更を示すタイムスタンプ。
- **isActive:** プロジェクトが現在アクティブかどうかを指定するブール値のフラグ。
- **タグ:** ビジュアライゼーションを分類するために使用されるオプションのフィールド。
- **data:** ビジュアライゼーションが表示するデータ

- [**CALENDAR**](../calendar/)

- [**MATRIX**](../matrix/)
