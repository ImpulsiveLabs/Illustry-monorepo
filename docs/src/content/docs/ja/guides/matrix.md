**マトリックス** ビジュアライゼーションは多次元のビジュアライゼーションであり、2 つのカテゴリのノードを相互に補完するものとして視覚化できます。

## 行列のデータ構造

行列を表すには、次の `NodeLinkData` インターフェイスを使用できます。

```typescript
{{Node}}

{{Link}}

{{NodeLinkData}}
```
### 主要な属性

- **name:** ノードの一意の識別子。
- **category:** ノードを特定のグループまたはクラスターに分類します。
- **labels:** は、ノードの特定の属性を表す名前値の配列です。
- **source:** リンクのソース ノードを識別します。
- **target:** リンクのターゲット ノードを識別します。
- **値:** ノードの接続の強さを示す、リンクに関連付けられた数値を表します。

### 長所と短所

#### 長所
- **大量のデータ:** 非常に大量のデータを 1 つのウィンドウに表示できます。

- **フィルタリング:** 行と列のフィルタリング。

#### 短所
- **スクロールが必要:** データが多すぎると、すべてを表示するにはスクロールが必要になる場合があります。
