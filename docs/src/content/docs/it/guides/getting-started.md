Illustry è una potente applicazione Web creata per fungere da hub di visualizzazione, integrando perfettamente varie visualizzazioni per diversi tipi di dati. Illustry dà vita all'esplorazione, alla tematizzazione e al filtraggio dei dati. Con un'interfaccia intuitiva e funzionalità avanzate, Illustry è progettato per soddisfare un'ampia gamma di utenti.

## Tecnologie chiave utilizzate

- **Frontend:** Next 16
- **Database:** MongoDB
- **Server:** Node

## Caratteristiche

- **Integrazione dati:** Supporta vari formati di file, tra cui JSON, XML, Excel e CSV.
- **Esplorazione dei dati:** Immergiti in profondità nei tuoi dati con potenti strumenti di esplorazione.
- **Temi:** personalizza facilmente l'aspetto delle tue visualizzazioni.
- **Filtro:** filtra e analizza facilmente i dati per estrarre informazioni significative.
- **Parco giochi:** esplora le funzionalità di Illustry nel parco giochi, dove puoi interagire con le funzionalità e testare diversi scenari.
