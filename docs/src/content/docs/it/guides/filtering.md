Il nostro strumento di visualizzazione introduce un linguaggio di sintassi analizzato, una bacchetta magica linguistica per la creazione di dati. 🚀

Illustry fornisce alcune costruzioni comuni e alcune parole specifiche affinché la sintassi venga analizzata correttamente:

### Costruzioni
- &&
- '>' (maggiore di)
- '<' (meno di)
- '=' (uguale)
- '>=' (maggiore o uguale)
- '<=' (minore o uguale)
- '!=' (non uguale)
