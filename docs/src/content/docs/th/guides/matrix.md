การแสดงภาพ **เมทริกซ์** คือการแสดงภาพหลายมิติ โดยที่โหนดของ 2 หมวดหมู่สามารถแสดงภาพเป็นส่วนเสริมกับอีกประเภทหนึ่งได้

## โครงสร้างข้อมูลเมทริกซ์

หากต้องการแสดงเมทริกซ์ คุณสามารถใช้อินเทอร์เฟซ `NodeLinkData` ต่อไปนี้:

```typescript
{{Node}}

{{Link}}

{{NodeLinkData}}
```
### คุณสมบัติที่สำคัญ

- **ชื่อ:** ตัวระบุที่ไม่ซ้ำกันสำหรับโหนด
- **หมวดหมู่:** จัดหมวดหมู่โหนดออกเป็นกลุ่มหรือคลัสเตอร์เฉพาะ
- **ป้ายกำกับ:** คืออาร์เรย์ของค่าชื่อที่แสดงถึงคุณลักษณะเฉพาะของโหนด
- **แหล่งที่มา:** ระบุโหนดต้นทางของลิงก์
- **target:** ระบุโหนดเป้าหมายของลิงก์
- **ค่า:** หมายถึงค่าตัวเลขที่เกี่ยวข้องกับลิงก์ที่แสดงให้เห็นว่าโหนดมีความเชื่อมโยงกันมากเพียงใด

### ข้อดีข้อเสีย

#### ข้อดี
- **ข้อมูลขนาดใหญ่:** ข้อมูลจำนวนมากสามารถแสดงได้ในหน้าต่างเดียว

- **การกรอง:** การกรองแถวและคอลัมน์

#### ข้อเสีย
- **จำเป็นต้องเลื่อน:** ข้อมูลมากเกินไปอาจต้องใช้การเลื่อนจึงจะเห็นทั้งหมด
