การแสดงภาพ **แผนภูมิวงกลม** เป็นวิธีที่กระชับและมีประสิทธิภาพในการนำเสนอการกระจายส่วนต่างๆ ภายในทั้งหมด มีประสิทธิภาพโดยเฉพาะอย่างยิ่งสำหรับการแสดงสัดส่วนและเปอร์เซ็นต์ในรูปแบบวงกลม


## โครงสร้างข้อมูลแผนภูมิวงกลม

หากต้องการแสดงแผนภูมิวงกลม คุณสามารถใช้อินเทอร์เฟซ `PieChartData` ต่อไปนี้:

```typescript
{{PieChartData}}
```
### คุณสมบัติที่สำคัญ

- **ค่า:** พจนานุกรมที่แต่ละคีย์เป็นหมวดหมู่หรือมิติข้อมูล และค่าที่เกี่ยวข้องเป็นค่าตัวเลข


### ข้อดีข้อเสีย

#### ข้อดี
- **การแสดงเปอร์เซ็นต์:** แผนภูมิวงกลมแสดงเปอร์เซ็นต์การมีส่วนร่วมของแต่ละหมวดหมู่ในภาพรวมอย่างชัดเจน

- **ความเรียบง่ายของภาพ:** รูปร่างวงกลมและความเรียบง่ายของแผนภูมิวงกลมทำให้ผู้ใช้เข้าใจได้ง่ายเพียงชำเลืองมอง

#### ข้อเสีย
- **จำกัดสำหรับหลายหมวดหมู่:** เมื่อต้องจัดการกับหมวดหมู่ต่างๆ มากมาย แผนภูมิวงกลมอาจหนาแน่นและยากต่อการตีความ
