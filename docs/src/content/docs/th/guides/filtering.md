เครื่องมือการแสดงภาพของเราแนะนำภาษาไวยากรณ์ที่แยกวิเคราะห์ ซึ่งเป็นไม้กายสิทธิ์ทางภาษาสำหรับการประดิษฐ์ข้อมูล 🚀

Illustry กำลังจัดเตรียมโครงสร้างทั่วไปและคำเฉพาะบางคำเพื่อให้แยกวิเคราะห์ไวยากรณ์ได้อย่างถูกต้อง:

### การก่อสร้าง
- &&
- '>' (มากกว่า)
- '<' (น้อยกว่า)
- '=' (เท่ากับ)
- '>=' (มากกว่าหรือเท่ากับ)
- '<=' (น้อยกว่าหรือเท่ากับ)
- '!=' (ไม่เท่ากัน)
