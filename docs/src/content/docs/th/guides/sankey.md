การแสดงภาพ **Sankey Diagram** เป็นเครื่องมือที่มีประสิทธิภาพสำหรับการแสดงภาพการไหลและความสัมพันธ์ระหว่างเอนทิตีในชุดข้อมูล มีประโยชน์อย่างยิ่งสำหรับการแสดงการกระจายและการเปลี่ยนแปลงของค่าทั่วทั้งระบบหรือกระบวนการ

## โครงสร้างข้อมูลไดอะแกรม Sankey

หากต้องการแสดง Sankey Diagram คุณสามารถใช้อินเทอร์เฟซ `NodeLinkData` ต่อไปนี้:

```typescript
{{Node}}

{{Link}}

{{NodeLinkData}}
```
### คุณสมบัติที่สำคัญ

- **ชื่อ:** ตัวระบุที่ไม่ซ้ำกันสำหรับโหนด
- **หมวดหมู่:** จัดหมวดหมู่โหนดออกเป็นกลุ่มหรือคลัสเตอร์เฉพาะ
- **แหล่งที่มา:** ระบุโหนดต้นทางของลิงก์
- **target:** ระบุโหนดเป้าหมายของลิงก์
- **ค่า:** หมายถึงค่าตัวเลขที่เกี่ยวข้องกับลิงก์ที่แสดงให้เห็นว่าโหนดมีความเชื่อมโยงกันมากเพียงใด

### ข้อดีข้อเสีย

#### ข้อดี
- **การแสดงโฟลว์:** แสดงภาพการไหลของค่าหรือปริมาณระหว่างเอนทิตีในระบบหรือกระบวนการอย่างมีประสิทธิภาพ

- **ข้อมูลเชิงลึกเกี่ยวกับการกระจาย:** ให้ข้อมูลเชิงลึกเกี่ยวกับวิธีการกระจายและเปลี่ยนแปลงค่าในแต่ละขั้นตอน

#### ข้อเสีย
- **ความซับซ้อนด้วยโหนดจำนวนมาก:** ความยุ่งเหยิงในการมองเห็นอาจเกิดขึ้นกับโหนดจำนวนมาก ซึ่งส่งผลต่อความสามารถในการอ่าน

- **กรณีการใช้งานเฉพาะทาง:** ออกแบบมาเพื่อการแสดงภาพการไหลและการกระจายเป็นหลัก อาจไม่เหมาะกับชุดข้อมูลทุกประเภท
