การแสดงภาพ **ช่องทาง** เป็นเครื่องมือที่มีประสิทธิภาพสำหรับการติดตามความก้าวหน้าและอัตราคอนเวอร์ชันของชุดขั้นตอนในกระบวนการ โดยให้การแสดงภาพว่าเอนทิตีเคลื่อนผ่านขั้นตอนต่างๆ ของเวิร์กโฟลว์ที่กำหนดไว้อย่างไร



## โครงสร้างข้อมูลช่องทาง

หากต้องการแสดงช่องทาง คุณสามารถใช้อินเทอร์เฟซ `FunnelData` ต่อไปนี้:

```typescript
{{FunnelData}}
```
### คุณสมบัติที่สำคัญ

- **ค่า:** พจนานุกรมที่แต่ละคีย์เป็นหมวดหมู่หรือมิติข้อมูล และค่าที่เกี่ยวข้องเป็นค่าตัวเลข


### ข้อดีข้อเสีย

#### ข้อดี
- **ตัวชี้วัดประสิทธิภาพ:** แผนภูมิช่องทางเหมาะอย่างยิ่งสำหรับการแสดงภาพตัวชี้วัดประสิทธิภาพ ช่วยให้ทีมสามารถติดตามและปรับปรุงตัวบ่งชี้ที่สำคัญได้

- **การวิเคราะห์แบบแบ่งกลุ่ม:** แต่ละขั้นตอนในช่องทางช่วยให้สามารถวิเคราะห์แบบแบ่งกลุ่ม ซึ่งช่วยระบุด้านที่ต้องปรับปรุงหรือประสบความสำเร็จโดยเฉพาะ

- **ข้อมูลเชิงลึกเชิงคาดการณ์:** แผนภูมิช่องทางสามารถให้ข้อมูลเชิงลึกเชิงคาดการณ์เกี่ยวกับประสิทธิภาพในอนาคตโดยพิจารณาจากอัตรา Conversion ในอดีต

#### ข้อเสีย

- **ความท้าทายกับกระบวนการที่ซับซ้อน:** ในสถานการณ์ที่กระบวนการเกี่ยวข้องกับสาขาหรือลูปที่ซับซ้อนจำนวนมาก แผนภูมิช่องทางอาจประสบปัญหาในการแสดงความซับซ้อนอย่างมีประสิทธิภาพ
