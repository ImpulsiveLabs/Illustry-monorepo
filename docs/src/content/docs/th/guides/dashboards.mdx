# แดชบอร์ดเป็นพื้นที่ทำงานแบบภาพ

แดชบอร์ดทำหน้าที่เป็นพื้นที่ทำงานภาพอเนกประสงค์ ซึ่งคุณสามารถจัดระเบียบและจัดการการแสดงภาพหลายรายการที่เกี่ยวข้องกับโครงการของคุณ โดยจัดให้มีสภาพแวดล้อมแบบไดนามิกและการโต้ตอบสำหรับการสำรวจ วิเคราะห์ และนำเสนอข้อมูลอย่างมีประสิทธิภาพ

## โครงสร้างข้อมูลแดชบอร์ด

หากต้องการแสดงแดชบอร์ดในระบบของคุณ คุณสามารถใช้อินเทอร์เฟซ `DashboardData` ต่อไปนี้:

```typescript
{{Layout}}

{{DashboardData}}

{{DashboardType}}

```
### คุณสมบัติที่สำคัญ

- **ชื่อ:** ตัวระบุที่ไม่ซ้ำสำหรับแดชบอร์ด
- **ชื่อโครงการ:** ชื่อของโครงการที่เกี่ยวข้องสำหรับการเชื่อมโยงตามบริบท
- **คำอธิบาย:** ช่องที่ไม่บังคับที่ให้บริบทหรือรายละเอียดเพิ่มเติมเกี่ยวกับแดชบอร์ด
- **createdAt:** การประทับเวลาที่ระบุเมื่อมีการสร้างแดชบอร์ด
- **updatedAt:** การประทับเวลาที่แสดงการแก้ไขแดชบอร์ดครั้งล่าสุด
- **การแสดงภาพ:** ชุดการแสดงภาพรวมอยู่ในแดชบอร์ด ไม่ว่าจะเป็นคู่คีย์-ค่าหรืออาร์เรย์ประเภทการแสดงภาพ
- **เลย์เอาต์:** รายการการกำหนดค่าเลย์เอาต์ที่กำหนดตำแหน่งและขนาดของการแสดงภาพภายในแดชบอร์ด แต่ละเลย์เอาต์ประกอบด้วย:
  - **i:** ตัวระบุที่ไม่ซ้ำกันสำหรับการแสดงภาพ
  - **x, y:** พิกัดที่แสดงตำแหน่งของการแสดงภาพบนตารางกริด
  - **w, h:** ความกว้างและความสูงของการแสดงภาพ
  - **minW, minH:** ข้อจำกัดความกว้างและความสูงขั้นต่ำ

### คุณสมบัติแดชบอร์ด

แดชบอร์ดช่วยให้ผู้ใช้สามารถ:

- ลากและวางการแสดงภาพเพื่อจัดเรียงใหม่ในพื้นที่ทำงาน
- ปรับขนาดการแสดงภาพให้เหมาะกับความต้องการข้อมูลเฉพาะ
- รักษาเลย์เอาต์หลายแบบเพื่อรองรับกรณีการใช้งานหรือกลุ่มเป้าหมายที่แตกต่างกัน

ด้วยการนำเสนอความยืดหยุ่นดังกล่าว แดชบอร์ดจึงช่วยให้ผู้ใช้ปรับแต่งพื้นที่ทำงานแบบเห็นภาพได้ตามความต้องการของโปรเจ็กต์
