# การแสดงภาพข้อมูลในพื้นที่ทำงานของคุณ

เมื่อทำงานกับการแสดงภาพ ระบบจะสนับสนุนโครงสร้างข้อมูลที่ยืดหยุ่นผ่านอินเทอร์เฟซ `VisualizationData`:

## โครงสร้างข้อมูลการแสดงภาพ

หากต้องการแสดงการแสดงภาพในระบบของคุณ คุณสามารถใช้อินเทอร์เฟซ `VisualizationData` ต่อไปนี้:

```typescript

{{VisualizationData}}

```

### คุณสมบัติที่สำคัญ
- **ชื่อโครงการ**: โครงการที่ใช้งานอยู่
- **ประเภท:** สตริงหรืออาร์เรย์สตริงที่ให้ประเภทของการแสดงภาพ (อาร์เรย์ใช้ได้กับ JSON's เท่านั้น)
- **ชื่อ:** ตัวระบุที่ไม่ซ้ำกันสำหรับการแสดงภาพ
- **คำอธิบาย:** ช่องที่ไม่บังคับเพื่อให้บริบทหรือรายละเอียดเพิ่มเติมเกี่ยวกับการแสดงภาพ
- **createdAt:** การประทับเวลาที่ระบุเมื่อมีการสร้างการแสดงภาพครั้งแรก
- **updatedAt:** การประทับเวลาซึ่งระบุการแก้ไขการแสดงภาพครั้งล่าสุด
- **isActive:** ธงบูลีนที่ระบุว่าโปรเจ็กต์มีการใช้งานอยู่หรือไม่
- **แท็ก:** ช่องที่ไม่บังคับซึ่งใช้เพื่อจัดหมวดหมู่การแสดงภาพ
- **ข้อมูล:** ข้อมูลที่การแสดงภาพจะนำเสนอ
