Illustry เป็นเว็บแอปพลิเคชันที่มีประสิทธิภาพซึ่งสร้างขึ้นเพื่อใช้เป็นศูนย์กลางการแสดงภาพ โดยผสานรวมการแสดงภาพต่างๆ สำหรับข้อมูลประเภทต่างๆ ได้อย่างราบรื่น Illustry นำการสำรวจข้อมูล การกำหนดธีม และการกรองมาสู่ชีวิต ด้วยอินเทอร์เฟซที่เป็นมิตรต่อผู้ใช้และคุณลักษณะขั้นสูง Illustry ได้รับการออกแบบมาเพื่อรองรับผู้ใช้ที่หลากหลาย

## เทคโนโลยีหลักที่ใช้

- **Frontend:** Next 16
- **Database:** MongoDB
- **Server:** Node

## คุณสมบัติ

- **การรวมข้อมูล:** รองรับรูปแบบไฟล์ต่างๆ รวมถึง JSON, XML, Excel และ CSV
- **การสำรวจข้อมูล:** เจาะลึกข้อมูลของคุณด้วยเครื่องมือสำรวจอันทรงพลัง
- **ธีม:** ปรับแต่งรูปลักษณ์และความรู้สึกของการแสดงภาพข้อมูลของคุณได้อย่างง่ายดาย
- **การกรอง:** กรองและวิเคราะห์ข้อมูลได้อย่างง่ายดายเพื่อดึงข้อมูลเชิงลึกที่มีความหมาย
- **สนามเด็กเล่น:** สำรวจความสามารถของ Illustry ใน Playground ซึ่งคุณสามารถโต้ตอบกับฟีเจอร์ต่างๆ และทดสอบสถานการณ์ต่างๆ ได้
