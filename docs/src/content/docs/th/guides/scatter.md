การแสดงภาพ **Scatter Plot** เป็นเครื่องมือที่มีประสิทธิภาพในการแสดงภาพความสัมพันธ์ระหว่างตัวแปรตัวเลข 2 ตัว ใช้จุดบนระนาบคาร์ทีเซียนเพื่อแสดงจุดข้อมูลแต่ละจุด ทำให้ง่ายต่อการระบุรูปแบบ ความสัมพันธ์ และค่าผิดปกติ


## โครงสร้างข้อมูลพล็อตกระจาย

หากต้องการแสดง Scatter Plot คุณสามารถใช้อินเทอร์เฟซ `ScatterData` ต่อไปนี้:

```typescript
{{ScatterPoint}}

{{ScatterData}}
```

### คุณสมบัติที่สำคัญ

- **ค่า:** อาร์เรย์ของค่าตัวเลข 2 ค่า [x, y] ซึ่งแสดงถึงพิกัดของจุดข้อมูลบนแกน X และ Y
- **ค่า:** สตริงที่ระบุหมวดหมู่หรือกลุ่มที่มีจุดข้อมูลอยู่


### ข้อดีข้อเสีย

#### ข้อดี
- **การระบุความสัมพันธ์:** แผนภูมิกระจายมีความเป็นเลิศในการเปิดเผยความสัมพันธ์ รูปแบบ และแนวโน้มระหว่างตัวแปรตัวเลข 2 ตัว

- **การตรวจจับค่าผิดปกติ:** ค่าผิดปกติหรือจุดข้อมูลที่เบี่ยงเบนไปจากค่าปกติอย่างมาก จะถูกระบุได้อย่างง่ายดายบน Scatter Plot

#### ข้อเสีย

- **การพล็อตโอเวอร์ที่เป็นไปได้:** ในกรณีที่จุดข้อมูลมีความหนาแน่นสูง อาจเกิดการโอเวอร์พล็อต ซึ่งส่งผลต่อการมองเห็นของแต่ละจุด
