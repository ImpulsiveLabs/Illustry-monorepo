Illustry นำเสนอ Data Playground อันทรงพลัง ซึ่งช่วยให้ผู้ใช้สามารถทดสอบและแสดงภาพข้อมูล JSON ของตนแบบอินเทอร์แอกทีฟก่อนที่จะบันทึก ฟีเจอร์นี้ออกแบบมาเพื่อมอบวิธีที่สะดวกและมีประสิทธิภาพในการดูว่าการแสดงภาพที่แตกต่างกันจะดูเป็นอย่างไรด้วยชุดข้อมูลเฉพาะของคุณ Playground ยังมีกลไกการตรวจสอบข้อมูลเพื่อให้แน่ใจว่า JSON ของคุณสอดคล้องกับโครงสร้างที่คาดหวัง
### การแสดงภาพแบบเรียลไทม์

Illustry Data Playground มอบการแสดงภาพข้อมูล JSON ของคุณแบบเรียลไทม์ เมื่อคุณป้อนหรือแก้ไขข้อมูล การแสดงภาพที่เกี่ยวข้องจะอัปเดตแบบไดนามิก ทำให้คุณทราบได้ทันทีว่าข้อมูลของคุณจะถูกนำเสนออย่างไร

## วิธีใช้ Data Playground

1. ไปที่ส่วน "Data Playground" ในแอปพลิเคชัน Illustry
2. วางหรือป้อนข้อมูล JSON ของคุณลงในโปรแกรมแก้ไขที่มีให้
3. เลือกประเภทของการแสดงภาพที่คุณต้องการทดสอบ
4. ดูการแสดงภาพแบบเรียลไทม์และปรับเปลี่ยนข้อมูลของคุณตามความจำเป็น
5. ตรวจสอบข้อความยืนยันเพื่อให้แน่ใจว่าข้อมูลของคุณมีรูปแบบที่ถูกต้อง
6. บันทึกข้อมูลของคุณหรือคัดลอก JSON ที่ผ่านการตรวจสอบเพื่อใช้ในโครงการของคุณ

## เหตุใดจึงต้องใช้ Data Playground

- **แสดงภาพได้ทันที:** ดูว่าข้อมูลของคุณจะมีลักษณะอย่างไรในการแสดงภาพต่างๆ โดยไม่จำเป็นต้องบันทึกหรือยอมรับการเปลี่ยนแปลง
- **การตรวจสอบข้อมูล:** ตรวจสอบให้แน่ใจว่าข้อมูล JSON ของคุณเป็นไปตามโครงสร้างที่คาดหวัง ช่วยลดข้อผิดพลาด และปรับปรุงขั้นตอนการทำงานของคุณ
- **การทดลอง:** ทดสอบชุดข้อมูลต่างๆ และสำรวจความสามารถของการแสดงภาพของ Illustry ในสภาพแวดล้อมที่ปราศจากความเสี่ยง
