การแสดงภาพ **กราฟเค้าโครงบังคับ** เป็นการแสดงข้อมูลที่เชื่อมโยงถึงกันแบบไดนามิกและน่าดึงดูด ใช้อัลกอริธึมโครงร่างที่บังคับทิศทางเพื่อถ่ายทอดความสัมพันธ์ระหว่างเอนทิตีภายในชุดข้อมูลด้วยภาพ

## โครงสร้างข้อมูลกราฟเค้าโครงบังคับ

หากต้องการแสดงกราฟรูปแบบบังคับ คุณสามารถใช้อินเทอร์เฟซ `NodeLinkData` ต่อไปนี้:

```typescript
{{Node}}

{{Link}}

{{NodeLinkData}}

```
### คุณสมบัติที่สำคัญ

- **ชื่อ:** ตัวระบุที่ไม่ซ้ำกันสำหรับโหนด
- **หมวดหมู่:** จัดหมวดหมู่โหนดออกเป็นกลุ่มหรือคลัสเตอร์เฉพาะ
- **แหล่งที่มา:** ระบุโหนดต้นทางของลิงก์
- **target:** ระบุโหนดเป้าหมายของลิงก์
- **ค่า:** หมายถึงค่าตัวเลขที่เกี่ยวข้องกับลิงก์ที่แสดงให้เห็นว่าโหนดมีความเชื่อมโยงกันมากเพียงใด

### ข้อดีข้อเสีย

#### ข้อดี
- **การแสดงภาพความสัมพันธ์:** แสดงให้เห็นความสัมพันธ์ระหว่างเอนทิตีอย่างชัดเจนผ่านโหนดและลิงก์ที่เชื่อมต่อกัน

- **เลย์เอาต์แบบไดนามิก:** ใช้เลย์เอาต์ที่บังคับทิศทางเพื่อการนำเสนอที่เป็นธรรมชาติและดึงดูดสายตา

#### ข้อเสีย
- **ความซับซ้อนด้วยกราฟหนาแน่น:** ในกราฟที่เชื่อมต่อกันอย่างหนาแน่น ภาพที่ไม่ชัดเจนอาจส่งผลต่อความสามารถในการอ่าน

- **จำกัดเฉพาะข้อมูลเครือข่าย:** มีประสิทธิภาพสูงสุดในการแสดงภาพเครือข่ายหรือข้อมูลตามความสัมพันธ์ อาจไม่เหมาะกับชุดข้อมูลทุกประเภท
