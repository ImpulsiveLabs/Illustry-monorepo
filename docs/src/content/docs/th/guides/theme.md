Illustry มีคุณลักษณะการปรับแต่งธีมที่มีประสิทธิภาพ ซึ่งช่วยให้ผู้ใช้ปรับแต่งรูปลักษณ์และความรู้สึกของการแสดงภาพได้ ไม่ว่าคุณจะชอบโหมดมืดหรือสว่าง คุณมีความยืดหยุ่นในการเลือกธีมที่กำหนดไว้ล่วงหน้าหรือปรับแต่งสีสำหรับการแสดงภาพแต่ละประเภท

## ธีมที่กำหนดไว้ล่วงหน้า

Illustry มาพร้อมกับชุดรูปแบบที่กำหนดไว้ล่วงหน้าซึ่งคุณสามารถนำไปใช้กับพื้นที่ทำงานของคุณได้อย่างง่ายดาย ธีมเหล่านี้ได้รับการออกแบบมาเพื่อให้แน่ใจว่าได้รับประสบการณ์ที่กลมกลืนและดึงดูดสายตา คุณสามารถสลับระหว่างธีมต่างๆ ได้อย่างง่ายดายเพื่อค้นหาธีมที่เหมาะกับความต้องการของคุณ

## การแก้ไขธีมที่กำหนดเอง

สำหรับผู้ใช้ที่ต้องการควบคุมความสวยงามของการแสดงภาพมากขึ้น Illustry เสนอคุณลักษณะการแก้ไขธีมที่กำหนดเอง ซึ่งจะทำให้คุณสามารถเลือกสีเฉพาะสำหรับองค์ประกอบต่างๆ ในการแสดงภาพแต่ละประเภทได้ ไม่ว่าจะเป็นการปรับสีพื้นหลัง เฉดสีของแผนภูมิ หรือเฉดสีของข้อความ คุณมีอิสระในการสร้างสรรค์ในการออกแบบพื้นที่ทำงานของคุณให้สมบูรณ์แบบ

## โหมดมืดและสว่าง

Illustry รองรับทั้งโหมดมืดและโหมดสว่าง ตอบสนองความต้องการที่แตกต่างกัน และรับประกันการมองเห็นที่เหมาะสมที่สุดในสภาพแวดล้อมที่หลากหลาย คุณสามารถสลับระหว่างโหมดต่างๆ ได้อย่างราบรื่นเพื่อปรับให้เข้ากับสภาพการทำงานของคุณ และปรับปรุงประสบการณ์ผู้ใช้โดยรวม

## วิธีการเข้าถึงฟีเจอร์ธีม

1. ไปที่ส่วน "ธีม" ในแดชบอร์ด Illustry
2. เลือกธีมที่กำหนดไว้ล่วงหน้าหรือป้อนตัวแก้ไขธีมที่กำหนดเอง
3. เลือกประเภทของการแสดงภาพที่คุณต้องการจัดธีม
4. ปรับสีและการตั้งค่าตามความต้องการของคุณ
