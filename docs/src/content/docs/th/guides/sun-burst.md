การแสดงภาพ **แผนภูมิ Sun Burst** เป็นการนำเสนอข้อมูลแบบลำดับชั้นแบบไดนามิกและน่าสนใจ ใช้เค้าโครงแบบรัศมีเพื่อถ่ายทอดลำดับชั้นขององค์ประกอบและความสัมพันธ์ภายในชุดข้อมูล


## โครงสร้างข้อมูลซันเบิร์สต์

หากต้องการแสดงข้อมูลแผนที่ คุณสามารถใช้อินเทอร์เฟซ `HierarchyData` ต่อไปนี้:

```typescript
{{HierarchyNode}}

{{HierarchyData}}
```
### คุณสมบัติที่สำคัญ

- **ชื่อ:** ชื่อโหนด
- **ค่า:** ตัวเลขที่แสดงถึงเมตริกเพื่อกำหนดขนาดของโหนดนั้น
- **หมวดหมู่:** อธิบายหมวดหมู่หรือประเภทที่โหนดนั้นอยู่
- **children:** คุณสมบัตินี้เป็นทางเลือกและแสดงถึงอาร์เรย์ของออบเจ็กต์ HierarchyNode มันบ่งบอกว่าโหนดสามารถมีโหนดลูกได้ โดยสร้างโครงสร้างแบบลำดับชั้น แต่ละโหนดย่อยมีโครงสร้าง HierarchyNode เดียวกัน ซึ่งช่วยให้สามารถแสดงลำดับชั้นที่เหมือนต้นไม้ได้

### ข้อดีข้อเสีย

#### ข้อดี
- **ความเข้าใจเชิงลำดับชั้น:** ซันเบิร์สต์มีความเป็นเลิศในการแสดงโครงสร้างข้อมูลแบบลำดับชั้น โดยให้ลำดับชั้นที่มองเห็นได้ซึ่งช่วยในการทำความเข้าใจความสัมพันธ์ระหว่างระดับต่างๆ

- **เค้าโครงแนวรัศมี:** เค้าโครงแนวรัศมีช่วยให้นำเสนอโครงสร้างแบบลำดับชั้นที่ดึงดูดสายตาและเป็นระเบียบ

#### ข้อเสีย
- **ความยุ่งเหยิงที่อาจเกิดขึ้น:** ในกรณีที่มีส่วนจำนวนมาก แผนภูมิอาจดูรก ส่งผลให้อ่านง่าย

- **จำกัดเฉพาะข้อมูลแบบลำดับชั้น:** ทรีแมปจะมีประสิทธิภาพมากที่สุดเมื่อแสดงข้อมูลแบบลำดับชั้น สำหรับชุดข้อมูลที่ไม่มีลำดับชั้น การแสดงภาพทางเลือกอาจมีความเหมาะสมมากกว่า
