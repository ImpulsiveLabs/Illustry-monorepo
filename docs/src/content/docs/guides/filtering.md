Illustry is providing some common constructions and some specific words in order for the syntax to be correctly parsed:

### Constructions
- &&
- '>' (greater than)
- '<' (less than)
- '=' (equal)
- '>=' (greater than or equal)
- '<=' (less than or equal)
- '!=' (not equal)
