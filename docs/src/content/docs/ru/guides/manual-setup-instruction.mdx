## Предварительные условия

Перед установкой Illustry Frontend убедитесь, что у вас есть следующие предварительные условия:

- Версия узла 18.17.0 или выше
- МонгоБД

## Illustry серверная часть


#### Локально:
В корне проекта необходим файл с именем `.env`, содержащий следующие поля:

```
MONGO_URL=mongodb:/127.0.0.1:27017/illustry
MONGO_TEST_URL=mongodb:/127.0.0.1:27017/illustrytest
MONGO_USER=<your_personal_mongo_user>
MONGO_PASSWORD=<your_personal_mongo_password>
ILLUSTRY_PORT=7000 
```
Выполните следующие команды:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Докер:

Выполните следующие команды:
```shell
docker build -t illustrybackend .
docker run -p 7000:7000 
```

## Illustry интерфейс


#### Локально:
В корне проекта необходим файл с именем `.env`, содержащий следующие поля:

```
NEXT_PUBLIC_BACKEND_PUBLIC_URL: http:/0.0.0.0:7000
```
Выполните следующие команды:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Докер:
**Перед запуском сначала создайте серверный образ!**

Выполните следующие команды:
```shell
docker build -t illustryfrontend .
docker run -p 3000:3000 -e NEXT_PUBLIC_BACKEND_PUBLIC_URL=http:/ilustrybackend:7000 illustryfrontend
```
