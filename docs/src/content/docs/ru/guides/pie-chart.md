Визуализация **Круговая диаграмма** – это краткий и визуально эффективный способ представить распределение частей в целом. Это особенно эффективно для отображения пропорций и процентов в круговом формате.


## Структура данных круговой диаграммы

Чтобы представить круговую диаграмму, вы можете использовать следующий интерфейс `PieChartData`:

```typescript
{{PieChartData}}
```
### Ключевые атрибуты

- **значения**. Словарь, в котором каждый ключ представляет собой категорию или измерение, а соответствующее значение – числовое значение.


### Плюсы и минусы

#### Плюсы
- **Процентное представление:** Круговые диаграммы дают четкое представление процентного вклада каждой категории в общее число.

- **Визуальная простота.** Круглая форма и простота круговых диаграмм позволяют пользователям легко понять их с первого взгляда.

#### Минусы
- **Ограничено для многих категорий.** При работе с многочисленными категориями круговые диаграммы могут оказаться перегруженными и сложными для интерпретации.
