Визуализация **Диаграмма Санки** — это мощный инструмент для визуализации потока и связей между сущностями в наборе данных. Это особенно полезно для демонстрации распределения и трансформации ценностей в системе или процессе.

## Структура данных диаграммы Санки

Чтобы представить диаграмму Санки, вы можете использовать следующий интерфейс `NodeLinkData`:

```typescript
{{Node}}

{{Link}}

{{NodeLinkData}}
```
### Ключевые атрибуты

- **имя:** Уникальный идентификатор узла.
- **категория:** Относит узел к определенной группе или кластеру.
- **source:** Идентифицирует исходный узел ссылки.
- **target:** Идентифицирует целевой узел ссылки.
- **value:** Представляет числовое значение, связанное со ссылкой, которое показывает, насколько сильно связаны узлы.

### Плюсы и минусы

#### Плюсы
- **Представление потока:** эффективно визуализирует поток значений или количеств между объектами в системе или процессе.

- **Информация о распределении:** дает представление о том, как ценности распределяются и трансформируются на каждом этапе.

#### Минусы
- **Сложность с многочисленными узлами.** При большом количестве узлов может возникнуть визуальный беспорядок, что влияет на читаемость.

- **Специальный вариант использования.** В первую очередь предназначен для визуализации потока и распределения; может не подходить для всех типов наборов данных.
