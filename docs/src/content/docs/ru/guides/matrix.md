Визуализация **Матрица** — это многомерная визуализация, в которой узлы двух категорий могут визуализироваться как дополняющие друг друга.

## Матричная структура данных

Для представления матрицы вы можете использовать следующий интерфейс `NodeLinkData`:

```typescript
{{Node}}

{{Link}}

{{NodeLinkData}}
```
### Ключевые атрибуты

- **имя:** Уникальный идентификатор узла.
- **категория:** Относит узел к определенной группе или кластеру.
- **labels:** Представляют собой массив значений имен, которые представляют определенные атрибуты узла.
- **source:** Идентифицирует исходный узел ссылки.
- **target:** Идентифицирует целевой узел ссылки.
- **value:** Представляет числовое значение, связанное со ссылкой, которое показывает, насколько сильно связаны узлы.

### Плюсы и минусы

#### Плюсы
- **Большие данные:** Очень большой объем данных может отображаться в одном окне.

- **Фильтрация.** Фильтрация по строкам и столбцам.

#### Минусы
- **Необходима прокрутка.** Чтобы просмотреть слишком много данных, может потребоваться прокрутка.
