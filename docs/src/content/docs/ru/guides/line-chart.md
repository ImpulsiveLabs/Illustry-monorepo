Визуализация **Линейная диаграмма** — это универсальный инструмент для представления тенденций и закономерностей в числовых данных с течением времени. Он широко используется для визуализации взаимосвязи между двумя непрерывными переменными и выделения тенденций или колебаний.

## Структура данных линейного графика

Чтобы представить линейный график, вы можете использовать следующий интерфейс `AxisChartData`:

```typescript
{{AxisChartData}}
```
### Ключевые атрибуты

- **заголовки**: массив строк, представляющих категории или измерения по одной из осей.
- **значения**. Словарь, в котором каждый ключ представляет собой категорию или измерение, а соответствующее значение представляет собой массив числовых значений вдоль другой оси.


### Плюсы и минусы

#### Плюсы
- **Многомерное сравнение.** Линейные диаграммы превосходно подходят для одновременного сравнения нескольких измерений, обеспечивая комплексное представление данных.

- **Четкое представление значений.** Числовые значения четко представлены по осям X и Y, что упрощает интерпретацию и сравнение пользователей.

#### Минусы
- **Сложность с чрезмерным количеством категорий.** При работе с большим количеством категорий или измерений визуализация может стать перегруженной и сложной для интерпретации.

- **Ограничено для категориальных данных.** Линейные диаграммы наиболее эффективны для числовых данных и могут быть не оптимальным выбором для представления категориальных данных.
