Наш инструмент визуализации представляет язык синтаксического анализа — лингвистическую волшебную палочку для обработки данных. 🚀

Illustry предоставляет некоторые общие конструкции и некоторые конкретные слова для правильного анализа синтаксиса:

### Конструкции
- &&
- '>' (больше чем)
- '<' (меньше чем)
- '=' (равно)
- '>=' (больше или равно)
- '<=' (меньше или равно)
- '!=' (не равно)

## Гистограмма/линейный график

### Принятые слова
- заголовки
- ценности

## Календарь

### Принятые слова
- категории
- даты

## Воронка/круговая диаграмма

### Принятые слова
- ценности

## Карта дерева / Солнечные лучи

### Принятые слова
- ценности
- категории

## Иерархическое объединение ребер/график с принудительной компоновкой/диаграмма Сэнки

### Принятые слова
- ценности
- категории
- цели
- имена

## Разброс

### Принятые слова
- категории
- xCoord
- yCoord

## Хронология

### Принятые слова
- типы
- даты
- авторы

## Облако слов

### Принятые слова
- ценности
