Illustry предлагает мощную игровую площадку для данных, позволяющую пользователям интерактивно тестировать и визуализировать свои данные JSON перед их сохранением. Эта функция создана для того, чтобы предоставить удобный и эффективный способ увидеть, как различные визуализации будут выглядеть с вашим конкретным набором данных. Игровая площадка также включает в себя механизм проверки данных, гарантирующий, что ваш JSON соответствует ожидаемой структуре.
### Визуализация в реальном времени

Площадка данных Illustry обеспечивает визуализацию ваших данных JSON в реальном времени. Когда вы вводите или изменяете данные, соответствующие визуализации динамически обновляются, предоставляя вам немедленную информацию о том, как будут представлены ваши данные.

## Как использовать игровую площадку данных

1. Перейдите в раздел «Площадка данных» в приложении Illustry.
2. Вставьте или введите данные JSON в предоставленный редактор.
3. Выберите тип визуализации, которую хотите протестировать.
4. Просматривайте визуализацию в реальном времени и при необходимости вносите коррективы в свои данные.
5. Просмотрите все сообщения проверки, чтобы убедиться, что ваши данные правильно отформатированы.
6. Сохраните свои данные или скопируйте проверенный JSON для использования в своих проектах.

## Зачем использовать игровую площадку данных?

- **Визуализация на лету.** Посмотрите, как ваши данные будут выглядеть в различных визуализациях, без необходимости сохранять или фиксировать изменения.
- **Проверка данных.** Убедитесь, что ваши данные JSON соответствуют ожидаемой структуре, что снижает количество ошибок и оптимизирует рабочий процесс.
- **Экспериментирование.** Протестируйте различные наборы данных и изучите возможности визуализации Illustry в безопасной среде.
