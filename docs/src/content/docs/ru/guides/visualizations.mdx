# Визуализации в вашем рабочем пространстве

При работе с визуализациями система поддерживает гибкую структуру данных через интерфейс `VisualizationData`:

## Структура данных визуализации

Чтобы представить визуализацию в вашей системе, вы можете использовать следующий интерфейс `VisualizationData`:

```typescript

{{VisualizationData}}

```

### Ключевые атрибуты
- **projectName**: активный проект.
- **тип:** Строка или массив строк, определяющий тип визуализации (массив работает только для JSON).
- **имя:** Уникальный идентификатор визуализации.
- **описание**. Необязательное поле для предоставления дополнительного контекста или сведений о визуализации.
- **createdAt:** Временная метка, указывающая, когда визуализация была первоначально создана.
- **updatedAt:** Метка времени, указывающая последнее изменение визуализации.
- **isActive:** Логический флаг, указывающий, активен ли проект в данный момент.
- **теги**: необязательное поле, используемое для категоризации визуализаций.
- **данные**: данные, которые будут представлены визуализацией.
