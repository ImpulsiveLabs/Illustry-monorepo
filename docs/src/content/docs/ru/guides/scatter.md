Визуализация **Точечная диаграмма** – это мощный инструмент для визуализации взаимосвязей между двумя числовыми переменными. Он использует точки на декартовой плоскости для представления отдельных точек данных, что упрощает выявление закономерностей, корреляций и выбросов.


## Структура данных точечной диаграммы

Для представления точечной диаграммы вы можете использовать следующий интерфейс `ScatterData`:

```typescript
{{ScatterPoint}}

{{ScatterData}}
```

### Ключевые атрибуты

- **значение:** Массив из двух числовых значений [x, y], представляющих координаты точки данных по осям X и Y.
- **значения**: строка, указывающая категорию или группу, к которой принадлежит точка данных.


### Плюсы и минусы

#### Плюсы
- **Идентификация взаимосвязей.** Диаграммы рассеяния превосходно подходят для выявления взаимосвязей, закономерностей и тенденций между двумя числовыми переменными.

- **Обнаружение выбросов.** Выбросы или точки данных, которые значительно отклоняются от нормы, легко выявляются на точечной диаграмме.

#### Минусы

- **Возможное перекрытие:** В случаях с высокой плотностью точек данных может возникнуть перекрытие, влияющее на видимость отдельных точек.
