# आपके कार्यक्षेत्र में विज़ुअलाइज़ेशन

विज़ुअलाइज़ेशन के साथ काम करते समय, सिस्टम `VisualizationData` इंटरफ़ेस के माध्यम से एक लचीली डेटा संरचना का समर्थन करता है:

## विज़ुअलाइज़ेशन डेटा संरचना

अपने सिस्टम में विज़ुअलाइज़ेशन का प्रतिनिधित्व करने के लिए, आप निम्नलिखित `VisualizationData` इंटरफ़ेस का उपयोग कर सकते हैं:

```typescript

{{VisualizationData}}

```

### प्रमुख गुण
- **प्रोजेक्टनाम**: सक्रिय प्रोजेक्ट।
- **प्रकार:** एक स्ट्रिंग या एक स्ट्रिंग सरणी जो विज़ुअलाइज़ेशन का प्रकार प्रदान करती है (सरणी केवल JSON के लिए काम करती है)।
- **नाम:** विज़ुअलाइज़ेशन के लिए एक विशिष्ट पहचानकर्ता।
- **विवरण:** विज़ुअलाइज़ेशन के बारे में अतिरिक्त संदर्भ या विवरण प्रदान करने के लिए एक वैकल्पिक फ़ील्ड।
- **createdAt:** टाइमस्टैम्प यह दर्शाता है कि विज़ुअलाइज़ेशन प्रारंभ में कब बनाया गया था।
- **updatedAt:** विज़ुअलाइज़ेशन के अंतिम संशोधन को दर्शाने वाला टाइमस्टैम्प।
- **सक्रिय है:** एक बूलियन ध्वज यह निर्दिष्ट करता है कि परियोजना वर्तमान में सक्रिय है या नहीं।
- **टैग:** विज़ुअलाइज़ेशन को वर्गीकृत करने के लिए उपयोग किया जाने वाला एक वैकल्पिक फ़ील्ड।
- **डेटा:** वह डेटा जो विज़ुअलाइज़ेशन प्रस्तुत करेगा
