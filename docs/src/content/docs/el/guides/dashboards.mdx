# Οι πίνακες εργαλείων ως οπτικοί χώροι εργασίας

Οι πίνακες ελέγχου λειτουργούν ως ευέλικτοι οπτικοί χώροι εργασίας όπου μπορείτε να οργανώσετε και να διαχειριστείτε πολλαπλές απεικονίσεις που σχετίζονται με τα έργα σας. Παρέχουν ένα δυναμικό και διαδραστικό περιβάλλον για την εξερεύνηση, την ανάλυση και την αποτελεσματική παρουσίαση δεδομένων.

## Δομή δεδομένων πίνακα ελέγχου

Για να αντιπροσωπεύσετε έναν πίνακα εργαλείων στο σύστημά σας, μπορείτε να χρησιμοποιήσετε την ακόλουθη διεπαφή `DashboardData`:

```typescript
{{Layout}}

{{DashboardData}}

{{DashboardType}}

```
### Βασικά Χαρακτηριστικά

- **όνομα:** Ένα μοναδικό αναγνωριστικό για τον πίνακα εργαλείων.
- **projectName:** Το όνομα του συσχετισμένου έργου για σύνδεση με βάση τα συμφραζόμενα.
- **περιγραφή:** Ένα προαιρετικό πεδίο που παρέχει πρόσθετο πλαίσιο ή λεπτομέρειες σχετικά με τον πίνακα εργαλείων.
- **createdAt:** Η χρονική σήμανση που υποδεικνύει πότε δημιουργήθηκε ο πίνακας εργαλείων.
- **updatedAt:** Η χρονική σήμανση που δείχνει την τελευταία τροποποίηση του πίνακα εργαλείων.
- **οπτικοποιήσεις:** Μια συλλογή οπτικοποιήσεων που περιλαμβάνονται στον πίνακα εργαλείων, είτε ως ζεύγη κλειδιών-τιμών είτε ως μια σειρά τύπων οπτικοποίησης.
- **διατάξεις:** Μια λίστα με διαμορφώσεις διάταξης που καθορίζουν τη θέση και το μέγεθος των απεικονίσεων εντός του πίνακα εργαλείων. Κάθε διάταξη περιλαμβάνει:
  - **i:** Ένα μοναδικό αναγνωριστικό για την οπτικοποίηση.
  - **x, y:** Συντεταγμένες που αντιπροσωπεύουν τη θέση της απεικόνισης στο πλέγμα.
  - **w, h:** Το πλάτος και το ύψος της απεικόνισης.
  - **minW, minH:** Ελάχιστοι περιορισμοί πλάτους και ύψους.

### Χαρακτηριστικά ταμπλό

Οι πίνακες ελέγχου επιτρέπουν στους χρήστες να:

- Σύρετε και αποθέστε απεικονίσεις για να τις αναδιατάξετε στον χώρο εργασίας.
- Αλλάξτε το μέγεθος των οπτικοποιήσεων ώστε να ταιριάζουν σε συγκεκριμένες ανάγκες δεδομένων.
- Διατηρήστε πολλαπλές διατάξεις για να καλύψετε διαφορετικές περιπτώσεις χρήσης ή κοινό.

Προσφέροντας τέτοια ευελιξία, οι πίνακες εργαλείων δίνουν τη δυνατότητα στους χρήστες να προσαρμόζουν τους οπτικούς χώρους εργασίας τους σύμφωνα με τις απαιτήσεις του έργου.
