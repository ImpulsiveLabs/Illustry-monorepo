Η οπτικοποίηση **Matrix** είναι μια πολυδιάστατη απεικόνιση, όπου οι κόμβοι 2 κατηγοριών μπορούν να οπτικοποιηθούν ως συμπληρωματικοί ο ένας με τον άλλο

## Δομή δεδομένων μήτρας

Για να αναπαραστήσετε ένα Matrix, μπορείτε να χρησιμοποιήσετε την ακόλουθη διεπαφή `NodeLinkData`:

```typescript
{{Node}}

{{Link}}

{{NodeLinkData}}
```
### Βασικά Χαρακτηριστικά

- **όνομα:** Ένα μοναδικό αναγνωριστικό για τον κόμβο.
- **κατηγορία:** Κατηγοριοποιεί τον κόμβο σε μια συγκεκριμένη ομάδα ή σύμπλεγμα.
- **ετικέτες:** Είναι ένας πίνακας τιμών ονομάτων που αντιπροσωπεύουν συγκεκριμένα χαρακτηριστικά του κόμβου.
- **πηγή:** Προσδιορίζει τον κόμβο προέλευσης του συνδέσμου.
- **target:** Προσδιορίζει τον κόμβο-στόχο του συνδέσμου.
- **τιμή:** Αντιπροσωπεύει μια αριθμητική τιμή που σχετίζεται με τη σύνδεση που υποδηλώνει πόσο ισχυρά συνδέονται οι κόμβοι.

### Τα υπέρ και τα κατά

#### Πλεονεκτήματα
- **Μεγάλα δεδομένα:** Πολύ μεγάλος όγκος δεδομένων μπορεί να εμφανιστεί σε ένα μόνο παράθυρο.

- **Φιλτράρισμα:** Φιλτράρισμα στις σειρές και στις στήλες.

#### Μειονεκτήματα
- **Απαιτείται κύλιση:** Πάρα πολλά δεδομένα μπορεί να χρειάζονται κύλιση για να τα δείτε όλα.
