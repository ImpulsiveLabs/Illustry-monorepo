---
title: Ξεκινώντας
description: Ένας οδηγός για Illustry
---

Το Illustry είναι μια ισχυρή εφαρμογή Ιστού που έχει κατασκευαστεί για να λειτουργεί ως κόμβος οπτικοποίησης, ενσωματώνοντας απρόσκοπτα διάφορες οπτικοποιήσεις για διαφορετικούς τύπους δεδομένων. Το Illustry ζωντανεύει την εξερεύνηση δεδομένων, τη δημιουργία θεμάτων και το φιλτράρισμα. Με φιλική προς το χρήστη διεπαφή και προηγμένες δυνατότητες, το Illustry έχει σχεδιαστεί για να εξυπηρετεί ένα ευρύ φάσμα χρηστών.

## Βασικές Τεχνολογίες που χρησιμοποιούνται

- **Frontend:** Next 16
- **Database:** MongoDB
- **Server:** Node

## Χαρακτηριστικά

- **Ενσωμάτωση δεδομένων:** Υποστηρίζει διάφορες μορφές αρχείων, όπως JSON, XML, Excel και CSV.
- **Εξερεύνηση δεδομένων:** Βουτήξτε βαθιά στα δεδομένα σας με ισχυρά εργαλεία εξερεύνησης.
- **Θέμα:** Προσαρμόστε την εμφάνιση και την αίσθηση των οπτικοποιήσεων σας με ευκολία.
- **Φιλτράρισμα:** Φιλτράρετε και αναλύστε εύκολα δεδομένα για να εξαγάγετε σημαντικές πληροφορίες.
- **Παιδική χαρά:** Εξερευνήστε τις δυνατότητες του Illustry στην παιδική χαρά, όπου μπορείτε να αλληλεπιδράσετε με τα χαρακτηριστικά και να δοκιμάσετε διαφορετικά σενάρια.
