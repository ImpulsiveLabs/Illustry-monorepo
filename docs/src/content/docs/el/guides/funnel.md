Η οπτικοποίηση **Διοχέτευση** είναι ένα ισχυρό εργαλείο για την παρακολούθηση της προόδου και των ποσοστών μετατροπών μιας σειράς σταδίων σε μια διαδικασία. Παρέχει μια οπτική αναπαράσταση του τρόπου με τον οποίο οι οντότητες κινούνται σε διαφορετικά στάδια μιας καθορισμένης ροής εργασίας.



## Δομή δεδομένων διοχέτευσης

Για να αναπαραστήσετε μια διοχέτευση, μπορείτε να χρησιμοποιήσετε την ακόλουθη διεπαφή `FunnelData`:

```typescript
{{FunnelData}}
```
### Βασικά Χαρακτηριστικά

- **τιμές:** Ένα λεξικό όπου κάθε κλειδί είναι μια κατηγορία ή μια ιδιότητα και η αντίστοιχη τιμή είναι μια αριθμητική τιμή.


### Τα υπέρ και τα κατά

#### Πλεονεκτήματα
- **Μετρήσεις απόδοσης:** Τα γραφήματα διοχέτευσης είναι εξαιρετικά για την οπτικοποίηση των μετρήσεων απόδοσης, δίνοντας τη δυνατότητα στις ομάδες να παρακολουθούν και να βελτιώνουν βασικούς δείκτες.

- **Τμηματοποιημένη ανάλυση:** Κάθε στάδιο στη διοχέτευση επιτρέπει την τμηματική ανάλυση, βοηθώντας στον εντοπισμό συγκεκριμένων περιοχών βελτίωσης ή επιτυχίας.

- **Προβλεπτικές πληροφορίες:** Τα γραφήματα διοχέτευσης μπορούν να παρέχουν προγνωστικές πληροφορίες για τη μελλοντική απόδοση με βάση τα ιστορικά ποσοστά μετατροπών.

#### Μειονεκτήματα

- **Πρόκληση με σύνθετες διεργασίες:** Σε σενάρια όπου οι διαδικασίες περιλαμβάνουν πολυάριθμους περίπλοκους κλάδους ή βρόχους, τα γραφήματα διοχέτευσης μπορεί να δυσκολεύονται να αναπαραστήσουν αποτελεσματικά την πολυπλοκότητα.
