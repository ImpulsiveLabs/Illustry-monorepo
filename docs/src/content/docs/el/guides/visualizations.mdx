# Οπτικοποιήσεις στον χώρο εργασίας σας

Όταν εργάζεστε με οπτικοποιήσεις, το σύστημα υποστηρίζει μια ευέλικτη δομή δεδομένων μέσω της διεπαφής `VisualizationData`:

## Οπτικοποίηση Δομή Δεδομένων

Για να αναπαραστήσετε μια οπτικοποίηση στο σύστημά σας, μπορείτε να χρησιμοποιήσετε την ακόλουθη διεπαφή `VisualizationData`:

```typescript

{{VisualizationData}}

```

### Βασικά Χαρακτηριστικά
- **ProjectName**: Το ενεργό έργο.
- **type:** Μια συμβολοσειρά ή ένας πίνακας συμβολοσειρών που παρέχει τον τύπο απεικόνισης (ο πίνακας λειτουργεί μόνο για JSON's).
- **όνομα:** Ένα μοναδικό αναγνωριστικό για την οπτικοποίηση.
- **περιγραφή:** Ένα προαιρετικό πεδίο για την παροχή πρόσθετου περιβάλλοντος ή λεπτομέρειες σχετικά με την οπτικοποίηση.
- **createdAt:** Η χρονική σήμανση που υποδεικνύει πότε δημιουργήθηκε αρχικά η οπτικοποίηση.
- **updatedAt:** Η χρονική σήμανση που υποδεικνύει την τελευταία τροποποίηση της απεικόνισης.
- **isActive:** Μια δυαδική σημαία που προσδιορίζει εάν το έργο είναι ενεργό αυτήν τη στιγμή.
- **ετικέτες:** Ένα προαιρετικό πεδίο που χρησιμοποιείται για την κατηγοριοποίηση των απεικονίσεων.
- **δεδομένα:** Τα δεδομένα που θα παρουσιάσει η οπτικοποίηση
