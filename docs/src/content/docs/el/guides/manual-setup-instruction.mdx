## Προαπαιτούμενα

Πριν εγκαταστήσετε το Illustry Frontend, βεβαιωθείτε ότι έχετε τις ακόλουθες προϋποθέσεις:

- Έκδοση Node 18.17.0 ή νεότερη
- MongoDB

## Illustry backend


#### Τοπικά:
Απαιτείται ένα αρχείο με το όνομα `.env` στη ρίζα του έργου που περιέχει τα ακόλουθα πεδία:

```
MONGO_URL=mongodb:/127.0.0.1:27017/illustry
MONGO_TEST_URL=mongodb:/127.0.0.1:27017/illustrytest
MONGO_USER=<your_personal_mongo_user>
MONGO_PASSWORD=<your_personal_mongo_password>
ILLUSTRY_PORT=7000 
```
Εκτελέστε τις παρακάτω εντολές:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Λιμενεργάτης:

Εκτελέστε τις παρακάτω εντολές:
```shell
docker build -t illustrybackend .
docker run -p 7000:7000 
```

## Illustry διεπαφή


#### Τοπικά:
Απαιτείται ένα αρχείο με το όνομα `.env` στη ρίζα του έργου που περιέχει τα ακόλουθα πεδία:

```
NEXT_PUBLIC_BACKEND_PUBLIC_URL: http:/0.0.0.0:7000
```
Εκτελέστε τις παρακάτω εντολές:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Λιμενεργάτης:
**Πριν εκτελέσετε, δημιουργήστε πρώτα την εικόνα του backend!**

Εκτελέστε τις παρακάτω εντολές:
```shell
docker build -t illustryfrontend .
docker run -p 3000:3000 -e NEXT_PUBLIC_BACKEND_PUBLIC_URL=http:/ilustrybackend:7000 illustryfrontend
```
