Η οπτικοποίηση **Scatter Plot** είναι ένα ισχυρό εργαλείο για την οπτικοποίηση των σχέσεων μεταξύ δύο αριθμητικών μεταβλητών. Χρησιμοποιεί σημεία σε ένα καρτεσιανό επίπεδο για να αναπαραστήσει μεμονωμένα σημεία δεδομένων, καθιστώντας εύκολο τον εντοπισμό προτύπων, συσχετίσεων και ακραίων τιμών.


## Δομή δεδομένων γραφήματος διασποράς

Για να αναπαραστήσετε ένα Scatter Plot, μπορείτε να χρησιμοποιήσετε την ακόλουθη διεπαφή `ScatterData`:

```typescript
{{ScatterPoint}}

{{ScatterData}}
```

### Βασικά Χαρακτηριστικά

- **τιμή:** Ένας πίνακας δύο αριθμητικών τιμών [x, y], που αντιπροσωπεύει τις συντεταγμένες ενός σημείου δεδομένων στους άξονες X και Y.
- **τιμές:** Μια συμβολοσειρά που υποδεικνύει την κατηγορία ή την ομάδα στην οποία ανήκει το σημείο δεδομένων.


### Τα υπέρ και τα κατά

#### Πλεονεκτήματα
- **Αναγνώριση σχέσης:** Οι γραφικές παραστάσεις διασποράς υπερέχουν στην αποκάλυψη σχέσεων, μοτίβων και τάσεων μεταξύ δύο αριθμητικών μεταβλητών.

- **Ανίχνευση ακραίων σημείων:** Τα ακραία σημεία, ή τα σημεία δεδομένων που αποκλίνουν σημαντικά από τον κανόνα, αναγνωρίζονται εύκολα σε ένα Σχέδιο Scatter.

#### Μειονεκτήματα

- **Πιθανή υπεργραφική παράσταση:** Σε περιπτώσεις με υψηλή πυκνότητα σημείων δεδομένων, μπορεί να προκύψει υπεργραφή, επηρεάζοντας την ορατότητα μεμονωμένων σημείων.
