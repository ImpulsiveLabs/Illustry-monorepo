Το εργαλείο οπτικοποίησης μας εισάγει μια αναλυμένη συντακτική γλώσσα, ένα γλωσσικό μαγικό ραβδί για τη δημιουργία δεδομένων. 🚀

Το Illustry παρέχει ορισμένες κοινές κατασκευές και ορισμένες συγκεκριμένες λέξεις προκειμένου η σύνταξη να αναλυθεί σωστά:

### Κατασκευές
- &&
- '>' (μεγαλύτερο από)
- '<' (λιγότερο από)
- '=' (ίσο)
- '>=' (μεγαλύτερο ή ίσο)
- '<=' (λιγότερο ή ίσο)
- '!=' (όχι ίσο)
