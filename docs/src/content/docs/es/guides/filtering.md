Nuestra herramienta de visualización presenta un lenguaje de sintaxis analizada, una varita mágica lingüística para la elaboración de datos. 🚀

Illustry proporciona algunas construcciones comunes y algunas palabras específicas para que la sintaxis se analice correctamente:

### Construcciones
- &&
- '>' (mayor que)
- '<' (menos que)
- '=' (igual)
- '>=' (mayor o igual)
- '<=' (menor o igual)
- '!=' (no igual)
