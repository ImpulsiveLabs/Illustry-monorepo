Illustry es una potente aplicación web creada para servir como centro de visualización, integrando a la perfección varias visualizaciones para diversos tipos de datos. Illustry da vida a la exploración, la tematización y el filtrado de datos. Con una interfaz fácil de usar y funciones avanzadas, Illustry está diseñado para atender a una amplia gama de usuarios.

## Tecnologías clave utilizadas

- **Frontend:** Next 16
- **Database:** MongoDB
- **Server:** Node

## Características

- **Integración de datos:** Admite varios formatos de archivo, incluidos JSON, XML, Excel y CSV.
- **Exploración de datos:** Profundice en sus datos con potentes herramientas de exploración.
- **Temática:** Personaliza la apariencia de tus visualizaciones con facilidad.
- **Filtrado:** Filtre y analice datos fácilmente para extraer información significativa.
- **Parque infantil:** Explora las capacidades de Illustry en el patio de juegos, donde puedes interactuar con las funciones y probar diferentes escenarios.
