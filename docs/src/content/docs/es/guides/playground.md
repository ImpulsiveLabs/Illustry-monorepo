Illustry ofrece un potente área de juegos de datos, que permite a los usuarios probar y visualizar interactivamente sus datos JSON antes de guardarlos. Esta característica está diseñada para proporcionar una manera conveniente y eficiente de ver cómo se verían las diferentes visualizaciones con su conjunto de datos específico. El área de juegos también incluye un mecanismo de validación de datos para garantizar que su JSON se ajuste a la estructura esperada.
### Visualización en tiempo real

Illustry Data Playground proporciona una visualización en tiempo real de sus datos JSON. A medida que ingresa o modifica los datos, las visualizaciones correspondientes se actualizan dinámicamente, brindándole información inmediata sobre cómo se presentarán sus datos.

## Cómo utilizar el campo de datos

1. Navegue a la sección "Data Playground" en la aplicación Illustry.
2. Pegue o ingrese sus datos JSON en el editor proporcionado.
3. Seleccione el tipo de visualización que desea probar.
4. Vea la visualización en tiempo real y realice ajustes a sus datos según sea necesario.
5. Revise los mensajes de validación para asegurarse de que sus datos tengan el formato correcto.
6. Guarde sus datos o copie el JSON validado para usarlo en sus proyectos.

## ¿Por qué utilizar el área de juegos de datos?

- **Visualice sobre la marcha:** Vea cómo se verán sus datos en diferentes visualizaciones sin la necesidad de guardar o confirmar cambios.
- **Validación de datos:** Asegúrese de que sus datos JSON se adhieran a la estructura esperada, reduciendo errores y optimizando su flujo de trabajo.
- **Experimentación:** Pruebe varios conjuntos de datos y explore las capacidades de las visualizaciones de Illustry en un entorno libre de riesgos.
