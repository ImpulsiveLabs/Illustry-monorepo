# Visualizaciones en su espacio de trabajo

Al trabajar con visualizaciones, el sistema admite una estructura de datos flexible a través de la interfaz `VisualizationData`:

## Estructura de datos de visualización

Para representar una visualización en su sistema, puede utilizar la siguiente interfaz `VisualizationData`:

```typescript

{{VisualizationData}}

```

### Atributos clave
- **nombredelproyecto**: el proyecto activo.
- **tipo:** Una cadena o una matriz de cadenas que proporciona el tipo de visualización (la matriz funciona solo para JSON).
- **nombre:** Un identificador único para la visualización.
- **descripción:** Un campo opcional para proporcionar contexto adicional o detalles sobre la visualización.
- **createdAt:** La marca de tiempo que indica cuándo se creó inicialmente la visualización.
- **updatedAt:** La marca de tiempo que indica la última modificación de la visualización.
- **isActive:** Un indicador booleano que especifica si el proyecto está actualmente activo.
- **etiquetas:** Un campo opcional utilizado para categorizar las visualizaciones.
- **datos:** Los datos que presentará la visualización.
