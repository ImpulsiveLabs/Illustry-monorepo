## Requisitos previos

Antes de instalar Illustry Frontend, asegúrese de tener los siguientes requisitos previos:

- Versión del nodo 18.17.0 o superior
- MongoDB

## Illustry servidor


#### En la zona:
Se necesita un archivo llamado `.env` en la raíz del proyecto que contiene los siguientes campos:

```
MONGO_URL=mongodb:/127.0.0.1:27017/illustry
MONGO_TEST_URL=mongodb:/127.0.0.1:27017/illustrytest
MONGO_USER=<your_personal_mongo_user>
MONGO_PASSWORD=<your_personal_mongo_password>
ILLUSTRY_PORT=7000 
```
Ejecute los siguientes comandos:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Estibador:

Ejecute los siguientes comandos:
```shell
docker build -t illustrybackend .
docker run -p 7000:7000 
```

## Illustry interfaz


#### En la zona:
Se necesita un archivo llamado `.env` en la raíz del proyecto que contiene los siguientes campos:

```
NEXT_PUBLIC_BACKEND_PUBLIC_URL: http:/0.0.0.0:7000
```
Ejecute los siguientes comandos:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Estibador:
**¡Antes de ejecutar, primero cree la imagen del backend!**

Ejecute los siguientes comandos:
```shell
docker build -t illustryfrontend .
docker run -p 3000:3000 -e NEXT_PUBLIC_BACKEND_PUBLIC_URL=http:/ilustrybackend:7000 illustryfrontend
```
