# Visualisierungen in Ihrem Arbeitsbereich

Bei der Arbeit mit Visualisierungen unterstützt das System eine flexible Datenstruktur über die Schnittstelle `VisualizationData`:

## Visualisierungsdatenstruktur

Um eine Visualisierung in Ihrem System darzustellen, können Sie die folgende `VisualizationData`-Schnittstelle verwenden:

```typescript

{{VisualizationData}}

```

### Schlüsselattribute
- **Projektname**: Das aktive Projekt.
- **Typ:** Ein String oder ein String-Array, das den Typ der Visualisierung bereitstellt (Array funktioniert nur für JSON).
- **Name:** Eine eindeutige Kennung für die Visualisierung.
- **Beschreibung:** Ein optionales Feld, um zusätzlichen Kontext oder Details zur Visualisierung bereitzustellen.
- **createdAt:** Der Zeitstempel, der angibt, wann die Visualisierung ursprünglich erstellt wurde.
- **updatedAt:** Der Zeitstempel, der die letzte Änderung der Visualisierung angibt.
- **isActive:** Ein boolesches Flag, das angibt, ob das Projekt derzeit aktiv ist.
- **Tags:** Ein optionales Feld, das zur Kategorisierung der Visualisierungen verwendet wird.
- **Daten:** Die Daten, die die Visualisierung präsentieren wird
