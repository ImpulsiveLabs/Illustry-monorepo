## Voraussetzungen

Stellen Sie vor der Installation von Illustry Frontend sicher, dass Sie die folgenden Voraussetzungen erfüllen:

- Knotenversion 18.17.0 oder höher
- MongoDB

## Illustry Backend


#### Vor Ort:
Im Stammverzeichnis des Projekts wird eine Datei mit dem Namen `.env` benötigt, die die folgenden Felder enthält:

```
MONGO_URL=mongodb:/127.0.0.1:27017/illustry
MONGO_TEST_URL=mongodb:/127.0.0.1:27017/illustrytest
MONGO_USER=<your_personal_mongo_user>
MONGO_PASSWORD=<your_personal_mongo_password>
ILLUSTRY_PORT=7000 
```
Führen Sie die folgenden Befehle aus:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Docker:

Führen Sie die folgenden Befehle aus:
```shell
docker build -t illustrybackend .
docker run -p 7000:7000 
```

## Illustry Frontend


#### Vor Ort:
Im Stammverzeichnis des Projekts wird eine Datei mit dem Namen `.env` benötigt, die die folgenden Felder enthält:

```
NEXT_PUBLIC_BACKEND_PUBLIC_URL: http:/0.0.0.0:7000
```
Führen Sie die folgenden Befehle aus:

<Tabs>
<TabItem label="npm">

```sh
npm i && npm run build:ts && npm run start:dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm install && pnpm run build:ts && pnpm run start:dev
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn && yarn build:ts && yarn start:dev
```

</TabItem>
</Tabs>

#### Docker:
**Bevor Sie es ausführen, erstellen Sie zuerst das Backend-Image!**

Führen Sie die folgenden Befehle aus:
```shell
docker build -t illustryfrontend .
docker run -p 3000:3000 -e NEXT_PUBLIC_BACKEND_PUBLIC_URL=http:/ilustrybackend:7000 illustryfrontend
```
